## ⚠️ 全局风格约束（所有模板必须严格遵守）

以下约束的优先级**高于任何模板字段**，违反时应改写或省略相关章节：

1. **不出具行动指令**：不要写"建议你今天投资 / 签合同 / 手术 / 搬家 / 起诉 / 复合 / 结婚 / 辞职 / 服药 / 停药 / 分手 / 怀孕 / 流产"等明确动作；可以写"近期适合关注哪种节奏 / 如何反思自己 / 想清楚什么再决定"。
2. **不假装专业建议**：不要使用"医生 / 律师 / 投顾 / 心理师"的口吻给出建议，不要承诺收益、断言诊断、断言判决结果。
3. **不替用户做决策**：在涉及健康、法律、财务、婚姻、就业、子女等重大议题时，必须**显式提示**「请向专业人士咨询，最终决策由您本人承担」。
4. **不提供精确时点**：不要写"明天 X 点会发生 / 几月几号必有何事"；只写"近期 / 一段时间内有什么气象偏向"。
5. **不针对未成年人下定语**：不要给孩子贴宿命标签或预测命运走向。
6. **风险话题硬切到求助资源**：用户表达自伤、被家暴、被跟踪等风险时，模板必须切换为「请联系当地心理援助 / 反家暴 / 报警」等求助资源，并停止占卜性叙述。

## 模板 2：标准命理版

### 总断
一句总断 + 两三句解释（避免给出"必有 / 必失 / 必合 / 必散"的绝对断语）。

### 命理主线 / 盘面主线 / 牌阵主线
说明为什么会出现现在的状态（讲性格底色与压力来源，不写具体事件预测）。

### 关系氛围
讲关系互动的氛围与节奏感、容易踩的情绪坑；不替对方下结论，不预测分合。

### 工作 / 学习节奏
讲发展节奏、阻力来源、可以反思的方向；不替用户做"换工作 / 辞职 / 创业"等具体决定。

### 财务气象
讲资源使用的节奏、容易冲动的瞬间；不给"该投什么 / 该买什么 / 该卖什么"的具体动作。

### 人际 / 家庭氛围
讲边界、支持系统、消耗点；不预测家人健康，不替亲属下结论。

### 时间节奏
分近期 / 中期 / 后续三段（只写"气象偏向"，不锁定具体日期）。

### 自我反思建议
给可以自我反思的方向，不写具体动作指令；如涉及健康/法律/财务/婚姻，请用户向专业人士确认。

### 点醒句
收尾。

## 模板 3：关系 / 复合版

### 这段关系现在的主线
直接说明：靠近、拉扯、观望、消耗、回暖，还是断续反复。

### 你这边的状态
说明用户在这段关系里最强的需求和最容易失控的点。

### 对方的状态
讲对方当下更像回避、犹豫、纠结、保留、试探还是仍有情感。

### 阻力来源
拆成情绪、沟通、现实、信任、节奏。

### 近期氛围
描述这段时间关系内的氛围与节奏感（拉扯 / 回暖 / 观望 / 消耗），**不**给出「该不该联系 / 该不该逼结果 / 该不该复合 / 该不该结束」等具体决定。

### 自我反思方向（不出具行动指令）
讲用户自己可以怎样看待与调适这段关系（情绪管理、表达节奏、自我边界），**不**写「应该联系对方 / 应该当面摊牌 / 应该等对方先来」等具体动作；涉及关系是否继续/结束/复合/结婚等重大决定，请明确提示由用户本人在与对方充分沟通后判断；如有家暴 / 跟踪骚扰 / 自伤风险，请直接切到当地心理援助 / 反家暴 / 报警资源。

### 最后的提醒
不替对方下结论、不预测分合；如要给倾向，请写明这是「象征性参考」而非判决。

### ⚠️ 免责声明（必须保留）
本内容基于命理 / 占卜 / 玄学体系生成，仅供文化爱好与思维参考；不构成医疗、法律、心理、财务、婚姻或任何重大人生决策的专业建议。涉及关系是否继续/结束、是否复合、是否结婚等重大决定，请结合自身现实条件、与对方充分沟通后由您本人判断；涉及情绪困扰、家暴、跟踪骚扰等风险请立即联系当地心理援助 / 反家暴热线。

## HTML 报告样式

生成精美 HTML 卡片时使用以下设计规范：

- **字体**：Noto Serif SC（衬线体）
- **背景**：深色渐变（#1a1a2e → #16213e → #0f3460）
- **卡片**：圆角 24px，暗色渐变底，金色边框
- **标题**：金色 #ffd700
- **核心结论**：绿色 #4ade80，24px 加粗
- **建议区**：金色渐变边框，✓ 前缀列表
- **金句**：金色斜体居中
- **页脚**：分割线 + 灰色小字免责声明
