# 分场大纲模板

用于 Step 4：生成分场大纲。

## 每场固定字段

| 字段 | 说明 |
|:--|:--|
| 场号 | 连续编号 |
| 场景标题 | 内/外 + 地点 + 日/夜 |
| 出场人物 | 本场出现的角色 |
| 本场戏剧任务 | 这场戏在叙事中完成什么 |
| 本场开始状态 | 人物关系/情绪进入时是什么样 |
| 核心动作/事件 | 本场发生的主要事件 |
| 情绪或关系转折 | 本场结束时和开始时有什么不同 |
| 必须出现的台词/物件/母题 | 本场必须保留的元素 |
| 场尾钩子/转场逻辑 | 本场如何结束，如何接下一场 |
| 备注 | 可压缩/不可压缩/可合并 |

## 可增加的核心写作提示（非必填）

只限必要护栏：
- 本场最容易写偏的风险；
- 一句必须保留的潜台词或动作；
- 禁止写法。

## 边界约束

分场大纲属于**编剧结构层**，不得提前写成分镜、导演阐述或演员读本。

- 不要写镜头号、景别、机位、运动、灯光方案、演员调度；
- 不要默认生成重场景卡；
- 长片或复杂故事按幕/段落组织；
- 短片可直接按 5-15 场组织。
