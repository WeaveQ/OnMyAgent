# 专业形态判断标准

用于 Step 1b：专业形态建议。

## 判断标准

### 适合长片

- 人物关系有多阶段推进；
- 时间跨度大；
- 物件/母题需要多次回收；
- 情绪靠延迟累积；
- 终局依赖前文细节沉淀；
- 强行压短会导致关系跳跃或情绪失真。

### 适合中短片

- 核心事件集中；
- 人物关系变化发生在少数几个场景内；
- 单一空间/单一夜晚/单一冲突即可完成主题表达；
- 物件链不复杂。

### 适合剧集

- 多人物、多支线、世界观/行业线可持续展开；
- 每集有独立推进和阶段性钩子；
- 主线关系不能在一部电影时长内充分展开。

### 适合短剧/微短剧

- 冲突高频、钩子密集、反转强；
- 场景短、人物目标外显；
- 如果故事主要依赖留白、克制和细节，**不应**强行改成短剧。

### 适合实验影像/艺术短片

- 故事事件少，但意象、声音、空间、身体动作或文本结构本身是核心表达。

## 语气要求

- 对明显不适配的形态，要直接说"不建议"，并说明代价；
- 不要用"都可以"敷衍；
- 允许提供备选，但必须有主推荐；
- 若用户坚持选择不推荐形态，尊重执行，但在提示包中写明"这是用户选择，非最佳适配"。

## 输出格式

- **推荐形态**：长片 / 中短片 / 剧集单集 / 短剧 / 微短剧 / 实验影像等；
- **推荐强度**：强推荐 / 可行 / 不建议；
- **理由**：从人物弧线、时间跨度、场景数量、关系层次、事件密度、商业目标、制作成本说明；
- **如果强行改成其他形态，会损失什么**；
- **用户仍可指定最终形态**，但需在提示包里记录该选择带来的改编代价。
