# 剧本正文排版规则

用于 Step 5a：生成标准剧本。

## 场标题

统一格式：

```md
## 场 1｜内景 培训班素描教室 - 下午
```

要求：
- 场号连续；
- 使用 `｜` 分隔场号与场景标题；
- 场景标题包含内/外景、地点、时间；
- 不要混用 `## 1.`、`### 第一场`、`场景一` 等格式。

## 动作说明

- 每段尽量短，便于手机阅读；
- 一段只写一个动作或一个清晰画面；
- 不把自检、分析、创作意图混入正文；
- 不写不可见心理，心理必须转成动作、停顿、物件或空间反应。

## 对白格式

人物名单独一行并加粗，下一行写对白：

```md
**姜宁**
时间到了，画笔放下。从第一排开始交卷。
```

不要写成：

```md
姜宁：时间到了，画笔放下。
```

也不要把人物名和动作说明混在一行。

## 画外音 / 屏幕文字 / 微信

画外音：

```md
**姜宁（画外）**
时间到了，画笔放下。
```

微信/短信等屏幕文字用引用格式：

```md
> **姜宁（微信）**
> 看完了。标注了23处。
```

若连续多条微信，可按角色分组，不要让聊天记录挤成大段。

## 文件组织

- 剧本正文文件只放剧本正文；
- 每一稿剧本完成后，必须另存对应版本自检报告；
- 阶段自检、写作说明、下一步建议，另存为 `项目名_剧本一稿自检报告.md` 等，或放在会话摘要里；
- 如果用户要求"工作稿"，可以保留简短元信息；如果用户要求"纯剧本"，删除所有说明性段落。
