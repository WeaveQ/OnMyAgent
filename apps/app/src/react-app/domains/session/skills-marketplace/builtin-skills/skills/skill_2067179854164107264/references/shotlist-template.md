# 分镜表模板与规则

用于 Step 6：分镜生成。

## 分镜层级

生成分镜前必须确认用户需要哪一种层级，避免把“关键分镜”误认为“完整逐镜头分镜”。

| 层级 | 名称 | 适用场景 | 输出密度 |
|:--|:--|:--|:--|
| A | 关键分镜表 | 测试、概念开发、视觉骨架、AI 方向初稿 | 覆盖全片/全季关键镜头，或每场 1 个关键镜头，重场可多拆 |
| B | 场级分镜表 | 拍摄执行初稿、较完整的导演/AI 视频准备 | 每场拆出主要镜头，但不追求最终逐镜头完整 |
| C | 逐镜头精细分镜 | 正式拍摄、AI 视频逐镜生成、重场执行 | 对指定场景/集数做完整镜头拆解 |
| D | 自定义范围 | 用户有明确需求 | 按用户指定场次、集数、镜头密度执行 |

默认规则：
- 用户未指定层级时，先询问选择，不直接生成；
- 用户说“简单分镜/关键镜头/先测试”→ 倾向 A；
- 用户说“完整一点/每场都拆”→ 倾向 B，但仍需确认；
- 用户说“完整分镜/逐镜头/拍摄用/AI 视频逐镜”→ 倾向 C；
- 长项目的 B/C 必须分批生成。

## 分镜表字段

每个镜头必须包含：

| 字段 | 必填 | 说明 |
|:--|:--|:--|
| 镜号 | ✅ | 本场内连续编号 |
| 对应场号 | ✅ | 关联剧本场号 |
| 景别 | ✅ | 全景/中景/近景/特写等 |
| 画面内容 | ✅ | 画面中正在发生什么 |
| 机位/角度 | | 正面/侧面/俯拍/仰拍等 |
| 镜头运动 | | 固定/推/拉/摇/跟/手持等 |
| 人物动作 | | 人物在画面中的主要动作 |
| 对白/声音 | | 本镜关键对白、环境声、音乐提示 |
| 道具/物件 | | 画面中需要出现的关键物件 |
| 情绪功能 | | 本镜在叙事中承担的情绪任务 |
| 时长估计 | 可选 | 预估秒数 |
| 执行备注 | 可选 | 拍摄注意事项、难点、备选方案 |

## 逐镜头精细分镜扩展字段（可选）

当用户选择 C「逐镜头精细分镜」，尤其用于正式拍摄或 AI 视频逐镜生成时，可在基础字段之外增加以下字段：

| 字段 | 说明 |
|:--|:--|
| 镜头目的 | 本镜在叙事、情绪、信息或节奏中的具体作用 |
| 起幅 | 镜头开始时的画面构图、主体位置、动作状态 |
| 落幅 | 镜头结束时的画面构图、主体位置、动作状态 |
| 主体运动 | 人物/物件/视线/空间关系在镜头内的变化 |
| 画面连续性 | 与前后镜头在方向、轴线、道具位置、伤痕、服装状态上的衔接 |
| 与上一镜衔接 | 切入方式、动作接续、视线接续、声音接续或情绪接续 |
| AI 视频提示词 | 可直接用于视频生成的画面描述，必须保持角色/场景/道具一致性 |
| 负面提示词 | 禁止生成的画面偏差，如换脸、年龄漂移、服装错误、甜宠化、过度暴力化等 |

扩展字段不是所有分镜都必填。只有在用户明确要求逐镜头、拍摄执行、AI 视频逐镜生成或镜头连续性控制时使用。

## 分镜生成原则

- 用户未确认分镜层级前，不要直接生成分镜；
- 分镜层级分为关键分镜表、场级分镜表、逐镜头精细分镜、自定义范围；
- 先按用户确认的层级和范围生成，不把关键分镜冒充完整逐镜头分镜；
- 长项目不要一次性输出超长全片逐镜头分镜，必须分批；
- 优先拆关键场景和高执行难度场景，但只有在用户选择关键分镜/重场分镜时才这样压缩；
- 文艺/克制关系戏要避免过度花哨镜头；
- 分镜必须服务剧本情绪，不替剧本补情节；
- 可以给镜头建议，但不要冒充最终导演方案；
- 若用户要求，可另行生成拍摄清单、道具清单、场景调度建议。

## 分镜输出节奏

关键分镜表：
- 可一次覆盖全片/全季视觉骨架；
- 但必须在文件说明中标注“关键分镜表，不是完整逐镜头分镜”。

场级分镜表：
- 长片/剧集建议按集、按幕或每次 5-15 场分批；
- 每批结束后标注完成到哪一场、下一批从哪一场开始。

逐镜头精细分镜：
- 建议每次 1-3 场；或
- 每次 20-40 个镜头；
- 每批结束后等待用户确认再继续。

## 分镜表格式示例

```md
| 镜号 | 场号 | 景别 | 画面内容 | 机位/角度 | 运动 | 人物动作 | 对白/声音 | 道具 | 情绪功能 | 时长 | 备注 |
|---:|---|---|---|---|---|---|---|---|---|---:|---|
| 1 | 50 | 全景 | 空教室，黄昏光 | 后方略低 | 固定 | 姜宁抱纸箱进入 | 环境声 | 纸箱 | 建立空旷感 | 6s | 先让空间压住人 |
```
