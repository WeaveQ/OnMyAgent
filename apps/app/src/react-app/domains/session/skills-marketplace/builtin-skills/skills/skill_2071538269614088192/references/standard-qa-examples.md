# 10 个标准用户问答（V1）

为 skill 提供可复用的标准回答样例，保持结构、口径和分流稳定。

## 使用方式

1. 把用户问题归到最接近的标准问答。
2. 保留"场景判断 → 功能推荐 → 操作路径 → 注意事项"结构。
3. 具体数字、对象、时效未说清时，补追问或提醒确认。

---

## QA1｜为什么孩子白天还能玩游戏？

**问法**："国家不是有限制吗？为什么我家孩子白天还能玩腾讯游戏？"

**回答**：场景 →"账号和规则为什么没按预期生效"

1. **P0 排查方向**
   - 孩子可能用的不是自己的未成年账号，而是成年人账号
   - 当前管控的账号不一定是孩子实际在用的账号
   - 账号可能未完成绑定/身份验证，或存在临时放行覆盖
2. **下一步**
   - 先确认账号来源（自己的/家长的/别人的）
   - 孩子自己的账号 → 检查【家庭管理】绑定和身份验证
   - 家长账号 → 优先看【人脸识别管控】或【绿色手机上报】
3. **注意**：未成年规则不能被放宽；成年人账号问题重点在解决账号来源

---

## QA2｜我想限制孩子每天玩游戏的时间

**问法**："我想限制孩子每天只能玩 1 小时，应该怎么设置？"

**回答**：场景 → 日常稳定管理

1. **P0**：【游戏时间限制】设置每天可玩时长或限定时段；可补【游戏充值限制】
2. **入口**：【游戏管理】→【时间限制】；新版对 AI 说"限制孩子每天玩1小时"
3. **前置**：孩子账号需先完成家庭绑定 + 身份验证
4. **注意**：未成年账号法定上限为仅限周五、周六、周日及法定节假日 20:00-21:00，成长守护只能在此基础上收紧，不能放宽；执行前确认对谁生效、全游戏还是指定游戏

---

## QA3｜我想限制孩子充值

**问法**："孩子老是乱充钱，我想限制充值，在哪里设置？"

**回答**：场景 → 消费风险控制

1. **P0**：【游戏充值限制】控制单次或周期充值额度
2. **P1**：风险高时补禁玩或更强账号安全能力
3. **入口**：【游戏管理】→【游戏充值限制】；新版对 AI 说"限制孩子充值"
4. **前置**：需先绑定并完成验证
5. **注意**：未成年消费上限不能突破健康系统规则；退款/申诉转家长柜台或服务与帮助

---

## QA4｜孩子拿我的手机玩游戏，怎么防止？

**问法**："孩子老拿我的手机玩游戏，我该开什么功能？"

**回答**：场景 → 孩子用家长账号玩游戏

1. **P0**：【人脸识别管控】/ 老版【家长自助下发人脸识别】
2. **P1**：补【守护锁】防改设置；用亲友微信账号的补【绿色手机上报】
3. **入口**：旧版首页直接可见；新版【管控工具】→【人脸识别管控】
4. **注意**：需家长身份核验；若只是临时借手机玩一会儿，更适合临时放行方案

---

## QA5｜我只想让孩子今天先玩半小时

**问法**："今天我先让他玩半小时，之后别再玩了，怎么弄？"

**回答**：场景 → 临时可控放行

1. **P0**：【临时锁】/【游戏临时开关】设临时放行 30 分钟
2. **P1**：同步加消费限制防误充；家长账号场景补守护锁或人脸识别
3. **入口**：旧版首页【游戏临时开关】；新版【管控工具】→【临时锁】；或对 AI 说"临时让他玩30分钟"
4. **注意**：临时规则可能覆盖原有限制；执行前确认对谁生效、是否全游戏、到期后是否恢复

---

## QA6｜怎么绑定孩子账号？

**问法**："我想开始管孩子，第一步怎么绑定？"

**回答**：场景 → 家庭绑定

1. **路径**：【腾讯成长守护】小程序 →【我的】→【家庭管理】→【绑定新成员】→ 孩子扫码/接受邀请
2. **绑定后**：可查询时长/充值；要设置限制还需补孩子身份验证
3. **建议**：绑定后先查一轮数据，再决定限时还是限充
4. **注意**：孩子不配合时转【家长服务助手】

## QA7｜守护锁有什么用？

**问法**："守护锁是干什么的？我需要开吗？"

**回答**：场景 → 防止孩子改设置/解绑

1. **解决问题**：防他人未经验证修改守护方案或解除家庭绑定
2. **适合谁**：已在用成长守护管理 + 担心孩子操作家长手机改规则
3. **入口**：【我的】→【守护锁】
4. **注意**：开关均需身份信息 + 人脸验证；"被他人开启"异常问题补热线/服务分流

---

## QA8｜怎么查孩子最近充了多少钱？

**问法**："帮我看看孩子最近有没有给游戏充钱，在哪里查？"

**回答**：场景 → 先查数据再决定管控

1. **P0**：查近 7 天/本月充值记录（总金额、订单明细、对应游戏）
2. **P1**：查完后可转【游戏充值限制】；风险高配合时间限制/禁玩
3. **入口**：游戏管理页面或对 AI 说"查询充值情况"；已绑定可看账号详情消费明细
4. **注意**：需先完成绑定；数据通常以近 30 天为主，太久可能查不到

---

## QA9｜我想禁掉王者荣耀

**问法**："不想让他玩王者荣耀，但别的游戏先不管，怎么设置？"

**回答**：场景 → 限制指定游戏

1. **P0**：【游戏禁玩】直接对目标游戏禁玩
2. **P1**：查该游戏时长/充值再决定禁多久；孩子可能换游戏时补总时长管理
3. **入口**：【游戏管理】→【游戏禁玩】；新版对 AI 说"把王者荣耀禁了"
4. **注意**："禁某款"≠"今天不准玩任何游戏"；执行前确认禁哪款、多久、对哪个账号

## QA10｜改实名 / 跳过人脸 / 解除防沉迷

**问法**："能不能改实名？" / "怎么跳过人脸？" / "怎么解除防沉迷？"

**回答**：场景 → 高风险规则边界，**不提供绕过方法**

1. **查实名状态** → 去【腾讯健康系统】小程序【实名认证查询】
2. **实名修改入口** → 以健康系统页面实际显示为准；未成年不支持随意修改
3. **人脸识别问题** → 由实名本人按页面提示完成；多次失败走官方服务咨询
4. **误绑/解绑** → 去【账号解除家庭关联申诉】或【服务与帮助】
5. **注意**：不提供"解除防沉迷""跳过人脸""违规改实名"方法；按问题类型分流到对应官方入口
