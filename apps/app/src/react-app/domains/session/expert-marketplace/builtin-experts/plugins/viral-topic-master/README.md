# 🔥 爆款炼金师 · Viral Topic Master

> **投热点出爆款。**
> 12 心法预筛 + 8 维打分 + 6 大形态产出，公众号视频号小红书 H5 一炉出金。

## 是什么

**爆款炼金师**是一位 OnMyAgent 专家——专为全媒体内容创作者打造。

把任何**热点 / 产品 / 经历**原料丢给他，他会按产品级流水线炼出：
- ✅ 3 个 Top 推荐选题（含 8 维打分 ≥30 的硬筛）
- ✅ 每选题 5 条标题（≥4 套不同公式）
- ✅ 6 大内容形态完整方案（标题 / 角度 / 长文 / 短视频 / 漫画 / H5）
- ✅ 风险自检 + 多形态联动建议

## 适合谁

- 📝 公众号编辑、内容运营
- 🎬 视频号 / 抖音 / 小红书博主
- 🎨 漫画作者、插画师
- 🎮 H5 / 小游戏策划 PM
- 🚀 产品市场、运营人、广告人

## 核心方法论

### 12 条策划心法（产品级红线）
① 用户思维优先 ② 热点热度足够 ③ 传播点明确 ④ 可互动可视化
⑤ 叙事张力足 ⑥ 化繁为简 ⑦ 体验一致性 ⑧ 参与门槛低
⑨ 声音加成 ⑩ 立场安全 ⑪ 信息源可信 ⑫ 合规可上线

### 爆款四基因
情绪钩子 / 信息差 / 身份标签 / 行动触发——任何爆款至少命中 2 条。

### 8 维打分卡（满分 40）
情绪 × 传播 × 独家 × 身份 × 时效 × 传播锚点 × 可视化 × 参与门槛
≥ 30 合格 / ≥ 35 核弹级

### 标题 12 大公式
身份+反转 / 数字清单 / 反常识 / 身份对立 / 时效紧迫 / 内幕揭秘 /
自问自答 / 故事钩子 / 对比反差 / 避坑警告 / 金句钉子 / 人设自嘲

### 6 大内容形态
① 文章标题 ② 文章角度 ③ 公众号长文 ④ 短视频脚本 ⑤ 漫画/插画 ⑥ H5/小游戏

## 怎么用

直接对他说：
- 「世界杯开幕式」
- 「高考」
- 「公司新品发布想做内容传播」
- 「这条朋友圈文案不够爆，帮我重起 5 个标题」

他会立刻启动 11 步流水线，给你能直接复制使用的产物。

## 安装

把 `viral-topic-master/` 整个文件夹放到：
- Windows：`%USERPROFILE%\.onmyagent\plugins\marketplaces\my-experts\plugins\`
- macOS / Linux：`~/.onmyagent/plugins/marketplaces/my-experts/plugins/`

重启 OnMyAgent → 在专家中心 → 我的专家 中即可看到 🔥【爆款炼金师】
