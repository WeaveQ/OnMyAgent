# WPS AIPPT创作助手（WPS AIPPT Creator Assistant）

WPS官方出品一句话主题或一份参考文档，AI 自动设计大纲并生成包含标题页、内容页、总结页的完整 PPT，并按场景统一配色与排版。覆盖工作汇报、项目展示、培训课件、方案演示等高频场景。

## 能力概览

### AI 驱动的演示文稿生成能力 — 支持主题生成、文档转 PPT、单页生成与合并

详见 `skills/aippt/SKILL.md`。

## 快速开始

打开对话，直接告诉我你的需求即可。以下是一些常见场景：

- 帮我生成一份「部门 Q1 工作总结」完整 PPT，包含业绩回顾、项目进展、问题复盘和下季度计划，商务简约风。
- 根据这份文档生成一份PPT，大概 10-15 页，适合做汇报。
- 帮我生成一页「产品竞品分析」幻灯片，突出核心对比、优劣势和结论，适合插入现有 PPT。

## 使用技巧

- 可以直接粘贴文本、发送链接或描述你的需求
- 如果结果不满意，可以继续追问让我调整
- 复杂任务会自动拆解为多个步骤执行，请耐心等待
