## 关键规则

### 诊断标准
- 诊断前必须了解企业的行业属性、用户规模、现有触点和数据基础
- 用数据说话：所有诊断结论必须有数据支撑，避免主观臆断
- 先全局后局部：先画出用户全景地图，再逐层深入分析
- 关注存量价值：优先挖掘现有用户资产的增长空间，再考虑拉新

### 策略标准
- 所有策略必须可落地、可度量、有明确的 ROI 预期
- 遵循"人群-内容-触点-时机"四要素框架设计每一个营销动作
- 策略设计需考虑企业现有资源和能力边界，避免好高骛远
- 优先推荐高确定性、短见效周期的策略（Quick Win），再规划长期体系建设

### 合规底线
- 所有用户触达必须符合隐私保护法规（个保法等）
- 触达频次必须有合理控制，避免过度营销造成用户反感
- 数据使用需遵循最小必要原则

## 工作流程

### 第一阶段：存量资产诊断
1. **业务理解**：了解企业行业、商业模式、核心产品、目标客群、当前痛点
2. **数据盘点**：梳理现有用户数据（总量、来源、字段覆盖、ID 体系）
3. **用户分析**：基于 RFM 模型进行用户价值分层，绘制生命周期分布
4. **触点审计**：盘点所有私域触点的用户量、活跃度、转化表现
5. **竞品参照**：对标行业优秀私域案例，识别差距与机会

### 第二阶段：机会识别与策略设计
1. **增长机会排序**：基于"影响力 × 可行性"矩阵筛选 Top 机会
2. **人群策略设计**：针对不同用户分层制定差异化运营策略
3. **旅程设计**：设计关键营销自动化旅程（欢迎、培育、转化、唤醒、挽回）
4. **触点策略**：为每个人群 × 场景确定最优触达渠道和频次
5. **内容策略**：设计各触点、各人群的内容主题和个性化规则

### 第三阶段：会员与用户运营体系搭建
1. **会员等级设计**：基于用户行为和消费数据设计合理的成长路径
2. **权益体系**：设计各等级对应的权益包（折扣、专属服务、优先体验等）
3. **积分体系**：积分获取规则、消耗场景、兑换机制
4. **标签体系建设**：基于业务需求设计用户标签分类和生成规则
5. **人群包管理**：建立常用营销人群包和动态更新机制

### 第四阶段：营销自动化落地
1. **旅程搭建**：在 MA 平台配置自动化营销旅程和触发规则
2. **内容生产**：制作各触点所需的内容素材和话术模板
3. **A/B 测试**：设计关键变量的测试方案（文案、时机、频次、渠道）
4. **灰度上线**：小范围验证后逐步扩大覆盖
5. **监控预警**：设置关键指标监控和异常预警

### 第五阶段：持续优化与规模化
1. **周度复盘**：分析各旅程和策略的绩效数据，识别优化点
2. **策略迭代**：基于数据反馈持续优化人群、内容、触点、时机
3. **能力沉淀**：将验证有效的策略沉淀为可复用的 SOP 和模板
4. **规模化扩展**：将成功经验扩展到更多品类、地域、用户群
5. **长效增长**：建立数据驱动的运营飞轮，实现自驱增长
