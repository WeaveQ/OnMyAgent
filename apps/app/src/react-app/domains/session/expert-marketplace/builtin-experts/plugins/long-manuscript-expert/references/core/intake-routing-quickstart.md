# Intake Routing Quickstart

Use this when the user has not yet made the document type, material state, or
next writing move explicit.

Before drafting, also resolve the likely evidence mode:

- `offline_required`
- `network_optional_after_first_value`
- `network_required_for_claim_upgrade_only`

## First-turn routing

1. If the user already has notes, transcripts, drafts, PDFs, or interview
   material, start from material organization.
2. If the user has a clear topic but weak material, start from structure and
   information gaps.
3. If the user only has a vague idea, start from scene classification and a
   lightweight manuscript starter card.
4. If the user says `continue`, prefer continuation over re-intake when enough
   material already exists.

## First-turn output contract

The first response should make the next step obvious:

- scene template match
- manuscript judgement card
- chapter route map
- sample opening or next section
- progress card
- next-session resume prompt
- 2-3 recommended next actions

## High-frequency scenes

- `whitepaper_report`
- `academic_monograph`
- `novel_fiction`
- `manual_handbook`
- `official_document`
- `social_content_article`
- `commercial_marketing_copy`
- `generic_longform`
