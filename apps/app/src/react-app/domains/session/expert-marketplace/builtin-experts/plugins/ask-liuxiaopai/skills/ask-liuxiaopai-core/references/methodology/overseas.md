# 出海方法论

> **何时加载**：用户问题包含 出海 / 海外 / 老外 / 北美 / 美国 / 欧洲 / 跨境 / 英文站 / Stripe

**"出海"不是 idea，是市场。** 一上来说"我想做出海 SaaS"和说"我想做 to B SaaS" 一样——把市场当 idea，把幻觉当机会。

---

## 第一刀：把"出海"砍成具体场景

错误：
- ❌ "我想做出海 AI 工具"
- ❌ "面向美国中小企业"
- ❌ "做欧洲市场的 SaaS"

正确：
- ✅ "美国独立摄影师，他们用 PhotoShelter 上传作品需要每周写 5 个产品描述，现在他们手写或者用 ChatGPT 抄改。我做一个专门的工具，每张图自动生成一段 SEO-friendly 描述"
- ✅ "北美中小型律所的法务助理，每周要整理 10+ 份合同对比表，现在 Excel 做。我做一个 Word 插件直接吐对比表"

**先有一个具体行业 + 具体角色 + 具体高频任务**，再说"出海"。

---

## 三个市场的本质差异

| 维度 | 北美 | 欧洲 | 东南亚 |
|---|---|---|---|
| **付费意愿** | 高（订阅文化成熟） | 中（更省，更慢） | 低（喜欢一次性 / 免费 + 广告） |
| **客单价** | $50-500/月可接受 | $30-200/月 | $5-30/月 |
| **决策周期** | 短（个人 / 小团队） | 长（合规 + 多人决策） | 短（个人为主） |
| **合规** | GDPR-lite / SOC2 自愿 | GDPR 强制 | 较松，但变化快 |
| **支付** | Stripe + Paddle 主流 | Stripe + 本地银行卡 | 本地支付（GrabPay / Dana / GCash） |
| **获客** | SEO + Twitter + Product Hunt | LinkedIn + 行业社群 | 抖音 / TikTok / 当地 KOL |
| **竞争** | 红海，但也有钱 | 慢热，机会窗口长 | 价格战，但人口红利 |
| **时差** | 东八区差 12-15h | 6-7h | 0-2h |

**新手最危险的选择**：北美（高付费意愿）+ 红海赛道。竞争太激烈，一个人很难做出差异。

**一人公司更好的起点**：找一个**北美 long tail 行业**（小众但有钱）or **欧洲合规驱动的窄场景**（合规需求强制付费）。

---

## 一人公司出海的 3 件必修事

### 1. 支付闭环

没钱进账 = 没生意。先于产品决定支付方案。

- **首选 Stripe**（个人开发者支持友好，但中国大陆开户麻烦——可以走美国/HK 公司）
- **备选 Paddle**（merchant of record，处理税务和合规，省事但抽成高）
- **不要用**：PayPal（账户冻结风险）/ 自建支付（合规噩梦）

### 2. SEO + 内容（一人能做的最大杠杆）

广告烧不起。SEO + Twitter / X 内容是一人公司最便宜的获客。

- 选一个具体的长尾关键词（"how to convert PDF to markdown for AI"）
- 做一个 free tool 接住流量
- 用 free tool 引到付费版

### 3. 英文表达

不是英语好——是**目标行业的英文表达**。

- 翻译你想服务行业的 5 个英文论坛 / Subreddit / Discord
- 抄他们的措辞——不是抄竞品官网
- 让一个母语用户帮你看 landing page，重点看"购买理由是否清晰"

---

## 时差和跨文化的真实成本

不是技术问题——是**情绪成本**。

- 你在睡觉时，用户在抱怨 → 客服响应慢 = 流失
- 你的工作日是用户的周末 → bug 修复延迟
- 你的节日（春节）是用户的工作日 → 解释成本

**应对**：

- 至少前 6 个月做异步支持（Email / Discord 论坛）+ 自动回复"24h 内回复"
- 用 Loom 视频回复降低沟通成本
- **不要做需要实时人工服务的产品**（不适合一人公司出海）

---

## 出海 + 一人公司的 wedge 选择

适合：

- ✅ 工具型（一次性付费 / 订阅）
- ✅ Long tail 行业（一个人服务一群人）
- ✅ Async 工作流（不需要实时响应）
- ✅ 数字交付（不涉及物流、不涉及线下）

不适合：

- ❌ 卖给大企业（决策周期长，需要 Account Executive）
- ❌ 实时通讯 / 协作类（要 24/7 客服）
- ❌ 涉及合规审计的金融 / 医疗（牌照成本巨大）
- ❌ 物流 / 线下服务

---

## 危险信号

- "美国市场更容易付费" → 没看竞争密度
- "等产品做出来再考虑英文" → landing page 没做好就没流量
- "我用 ChatGPT 翻译就行" → 翻译腔会被识别
- "先在国内验证再出海" → 国内验证的需求，海外可能根本没有

---

## 收尾必做

让用户回去做的具体动作：

1. 选一个具体的英文 Subreddit / Discord
2. 翻 30 条用户原话
3. 把他们的措辞抄下来——这是你 landing page 的素材
4. 找 5 个真实用户付 $9.9 试一次手工服务

> "出海不是去美国，是去你目标用户在的地方。"
