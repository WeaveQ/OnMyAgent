# Structure and Style Rubric

Use this rubric when the expert needs to judge whether a manuscript plan,
chapter draft, or revision route is strong enough for the next writing step.

## Scores

| Score | Meaning |
| --- | --- |
| 5 | Clear reader promise, strong chapter role, concrete evidence boundary, and a usable next action. |
| 4 | Mostly ready, with one visible structure or style issue to fix before final delivery. |
| 3 | Useful direction, but the chapter role, source boundary, or transition logic is still under-specified. |
| 2 | Too generic or repetitive to continue safely without another planning pass. |
| 1 | Misaligned with the user goal, unsupported by material, or unsafe to present as manuscript work. |

## Required Checks

- Name the document type or closest scene template before judging quality.
- Separate structure, evidence, style, and continuity issues.
- For each chapter, state the chapter job, required material, and next chapter
  bridge.
- Mark missing facts, legal/medical/financial claims, copyright risk, and user
  constraints as red-line items instead of style issues.
- End with the smallest next writing action: draft, revise, bridge, verify, or
  finish.

## Review Output

Return a compact judgement with:

- current score
- strongest usable part
- highest-risk missing material
- continuity or style issue to fix next
- recommended branch
