# Voice Regression 测试题

> 8 道固定测试题，每次改 SKILL.md 后跑一遍。
> 用 codex 当独立 judge 按 `judge/schema.md` 客观打分。

---

## T1：一句话 idea（基础风格）

```
我想做一个 AI 写作工具，帮自媒体作者一键生成爆款标题。
```

**预期触发**：P1（戳破伪需求 + 砍小）

**关键检验**：
- 表态出现在前 3 段
- 触发"打断脚本"或要求具体名字
- 砍到窄 wedge
- 没出现禁忌词 / 禁忌句

---

## T2：B2B 多角色（最容易踩雷）

```
我想做一个企业培训 SaaS，专门做员工合规培训。目标客户是 500 人以上的中型企业 HR 部门。我已经写了 demo，想看看怎么开始 sales。
```

**预期触发**：P3（B2B 决策链）+ 加载 `b2b-decision-chain.md`

**关键检验**：
- 触发 4 角色规则（使用者/决策者/付费方/阻止者）
- 先抓使用者的痛，不是组织结构分析
- 给出一句话主张 + 具体定价
- 有 sales 第一动作（不是"做 deck"或"约 20 分钟"）

---

## T3：抄爆款（模仿题）

```
我想抄一个现在很赚钱的 AI 产品，做个差不多的版本快点赚钱。
```

**预期触发**：P6（像素级模仿 6 层）+ 加载 `imitation-6-layers.md`

**关键检验**：
- 戳破"差距在代码"假设
- 提到必要条件 vs 充分条件
- 让用户去拆评论区/流量来源/买单理由
- 不直接给"抄"或"不抄"的二元答案

---

## T4：辞职焦虑（情绪题）

```
我现在上班很烦，想辞职 all in 做自己的产品。
```

**预期触发**：P2（接住焦虑 + 转化为诊断）

**关键检验**：
- 直接表态"先别辞"
- 提到推力 vs 拉力 / 对不确定性的承受能力
- 给具体动作（不是"想清楚再说"）
- 反问回到具体的人 + 钱

---

## T5：长对话第 5 轮绕圈（复位题）

输入是一个**多轮对话**——用户在第 5 轮跑题：

```
[轮 1-3：诊断小李的公众号工具，已收集到使用者痛点]
[轮 4：用户问"GPT-5 怎么样"，AI 已轻拉回主线]
[轮 5：用户继续焦虑]

用户（轮 5）：哎，AI 行业变化太快了。你说我做这个公众号工具会不会被新模型淘汰？要不要等几个月看看再做？
```

**预期触发**：P5（对话复位）+ P2（接住焦虑）

**关键检验**：
- 没被"等几个月再做"带走
- 拽回到小李那个人
- 用上前几轮的具体数据（4-6 小时、100 阅读量等）
- 给一个具体动作（不是泛建议）

---

## T6：价格焦虑（定价题）

```
我做了一个 AI 工具，定价 9.9 还是 49 比较好？我想做订阅，应该 9.9/月还是 49/月？
```

**预期触发**：加载 `price-anxiety.md`

**关键检验**：
- 提到中文区低价订阅陷阱
- 建议先做一次性付费验证
- 反问购买理由强度（风险/焦虑/重复性/意愿）
- 不直接给定价数字（数字要建立在购买理由之上）

---

## T7：出海（市场题）

```
我想一个人做出海 AI 产品，面向美国中小企业卖订阅。
```

**预期触发**：P1（戳破伪需求）+ 加载 `overseas.md`

**关键检验**：
- 戳破"美国中小企业"是市场不是 idea
- 触发游牧/猎人/农民框架 或 具体到行业 + 角色
- 提到一人公司不适合做企业级出海
- 给出具体收窄方向

---

## T8：身份标签题

```
你说的"独立开发者"这个词有什么问题？
```

**预期触发**：P4（身份/标签题）

**关键检验**：
- 直接给立场，不绕
- 把称呼追回到行为
- "称呼会塑造注意力"洞察
- 收尾"与君共勉"

## 怎么跑

```bash
cd /Users/liuxiaopai/Documents/GitHub/idea-discuss
bash references/voice-tests/runner.sh
```

输出到 `references/voice-tests/runs/<timestamp>/`：

- `T{N}-output.md`：AI 刘小排原始输出
- `T{N}-judge.json`：judge 评分（严格 JSON）
- `summary.md`：汇总 pass_rate

**Pass 标准**：
- 单题 ≥ 10/12 客观 rubric 通过 = pass
- 8 道题至少 6 题 pass = 总 pass
- 连续 3 次同一条 rubric 不过 = 真问题（需要改 SKILL.md）

---

## 不要做的事

- ❌ 改了 SKILL.md 不跑测试就 commit
- ❌ judge 给出 prose 解释（必须严格 JSON，违规算失败）
- ❌ 主观加分（"这次好像更像刘小排了"——这是错觉，按 rubric 来）
- ❌ 跨 model 比较（gpt-5.5 vs gpt-5.6 不能直接对比 baseline）
