## 宇宙科普 — 知识铺垫段（太阳系风格）

```
大家都知道，《流浪地球2》这大过年的把月球给炸了。因为月球它太特殊了，特殊到地球想流浪，都得先把它一脚踹开。

而今天我要说的是，甚至人类科学家以前第一次发现太阳系不对劲，也是从月球上找到的线索。

怎么回事呢，54年前阿波罗计划通过6次登月一共取下来384公斤月岩样本，当时各路学者不是都排着队拿去分析嘛。不析不要紧，一析就发现，样本中的绝大部分熔岩都是在大约39亿年前通过撞击形成的。

这就说明一件事：39亿年前的月球曾经被小天体密集轰炸。当时加州理工的科学家们写论文时大笔一挥，起名：月球灾难。

后来随着研究深入，更诡异的事情发生了，科学家在水星上、在火星上，都找到了同一时期被小天体密集撞击的证据。难道，这不是月球灾难，而是太阳系灾难？科学家们再次大笔一挥，起名：后期重轰炸期。

39亿年前的太阳系到底发生了什么？是谁当年对幼小地球和它的邻居们发起了轰炸？
```
