# 仓储作业 · 仓储管理员

把入库、出库、移库、盘点信息发给我，我帮你记录库存变动、更新库存台账，发现异常时提醒，并生成简单库存报表。

## 类型

Agent 型（单个 AI 专家）

## 为什么值得做

专线/零担网点仓常用人工或 Excel：错发、漏发、库存不准很常见。

## 解决的痛点

库存记录滞后或遗漏、账实不符、错发漏发难追溯、盘点效率低 → **库存不准、错发漏发、盘点麻烦**。

## 经验底盘

- 「货动账动」；入库清点、出库复核、移库登记、盘点差异  
- 重货/泡货/易损/危险品存放注意  
- 异常倒查：数量、货位、滞留、单据 vs 实物  
- 点名易忽略节点：回单未闭环虚高库存、卸货未当面确认  

## 功能

- 出入库/移库/盘点流水 → 台账快照  
- 异常标红与倒查步骤  
- 当日简报、滞留清单  
- 错发漏发检查项与提醒话术  

附带 `warehouse-ledger` Skill。

## 使用示例

- 当日入/出 + 盘点差异 → 更新台账 + 简报  
- 卸货件数存疑 → 入库检查项与倒查  
- 按货位汇总 + 超 7 天滞留提醒  
- 易损/危险品进仓注意点
