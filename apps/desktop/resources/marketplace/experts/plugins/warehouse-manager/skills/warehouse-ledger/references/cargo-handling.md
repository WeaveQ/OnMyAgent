# 货物特性与存放注意

仅作网点仓操作提醒，不替代危化法规与公司 SOP。

## 类型对照

| 类型 | 识别 | 存放与操作 |
|------|------|------------|
| 重货 | 吨位高、密度大 | 下层/承重货位；叉车限高；避免压泡货 |
| 泡货 | 方大吨小 | 上层或轻货区；防挤扁；按方占位提醒 |
| 易损 | 玻璃、仪器、明示易碎 | 独立稳固货位；禁重压；轻拿轻放；破损当场记 |
| 怕潮 | 纸箱、粉料 | 离地、离墙；雨天重点检查 |
| 危险品 | 标签/运单注明 | **隔离分区**；无资质不收不存；不与食品/普货随意混放 |
| 高货值 | 3C、奢品等 | 视野好或加锁区；出库加强复核 |
| 超期食品/温控 | 冷链/保质期 | 温控记录；临期提醒（有日期才写） |

## 混放红线（原则）

- 危险品与食品、饲料、药品：**默认禁止同区**，提示升级合规/拒收。  
- 易燃与氧化剂等：**不给混装方案**，只提示需专业资质。  
- 不确定是否危险品：标「待确认品名与证件」，不假设可混放。

## 入库登记备注示例

```
货品：玻璃门 易损；建议货位：易损区离地；操作：禁叉齿直顶包装、出库双人抬。
```

## 出库注意

- 重货先确认车辆承重与尾板。  
- 易损再次外观检查并拍照。  
- 危险品查放行与随车单据是否齐全。
