# 责任初判逻辑（作业级，非法律意见）

## 原则

1. **先事实后责任**：时间线与证据不齐时，只给「可能方向 + 待证点」。  
2. **多因一果常见**：包装 + 装卸 + 在途可并存，写清权重不确定。  
3. **免责不默认成立**：需条款与事实支撑（如托运违禁、包装明示不足且可证明）。  
4. **禁止写死**「法院必判某某全责」。

## 检查维度

| 维度 | 关注点 | 倾向（仅参考） |
|------|--------|----------------|
| 包装 | 是否符合货品、有无木架/填充、托运是否特殊包装要求 | 包装明显不达标 → 托运方责任权重上升 |
| 装卸 | 是否野蛮装卸、叉车戳损、高空摔落 | 装卸不规范 → 操作方责任权重上升 |
| 在途节点 | 碰撞、渗漏、转驳异常、密封失效 | 在途异常 → 承运环节责任权重上升 |
| 交接批注 | 签收是否注明异常、是否事后开箱 | 无批注事后主张 → 举证难度上升（双方） |
| 延误 | 约定时效、是否不可抗力、是否客户等通知/地址错误 | 分情形 |
| 丢件 | 最后清晰交接节点 | 断点所在环节重点排查 |
| 保价/限额 | 有无保价、限额条款 | 影响赔付路径，不等于无责 |

## 输出格式

```markdown
### 责任初判（对内，非法律意见）
- 可能方向：…
- 主要依据：…
- 反证/免责关注：…
- 证据缺口：…
- 建议策略：先补证 / 部分协商 / 报案保司 / 升级法务
- 置信度：高 / 中 / 低
```

## 对外表述约束

- 未获老板授权：**不写「我司全责」「一定赔」**。  
- 可用：「正在核查节点与证据，将于 {时限} 前反馈处理方案」。
