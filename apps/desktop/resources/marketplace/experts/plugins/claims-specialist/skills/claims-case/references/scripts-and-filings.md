# 沟通话术与报案提纲

## 1. 客户（安抚 + 边界）

```
{称呼}，非常理解您着急。运单 {waybill} 我们已登记异常（{简述}），正在核查装/卸/在途节点与影像资料。
请您先协助保留现场货物与包装，并提供{货值证明/损失清单}。
我们会在 {时限，如24-48小时} 内给您阶段反馈，不让您空等。
（核实完成前，具体赔付方案需以事实与约定为准，我们会尽快推进。）
```

## 2. 客户（方案沟通，需你定金额边界后改）

```
{称呼}，核查进展：{已确认事实}。尚待确认：{缺口}。
结合运单约定与目前材料，我们建议下一步：{协商部分承担 / 走保险 / 补充鉴定}。
请您看是否接受{方案要点}；有分歧我们继续对齐材料，争取尽快结案。
```

## 3. 司机 / 承运问询

```
{师傅/公司}，运单 {waybill} 出现{异常}。请书面回复：
1）装车时外观与件数
2）在途是否转驳/碰撞/雨淋
3）卸车交接谁签、有无批注
4）相关照片/监控线索
今日内回复便于对客与报案，谢谢。
```

## 4. 保司报案（要素提纲）

```
【报案说明】
- 被保险人/报案人：
- 保单号/险种：（待填）
- 运单号/线路/货品：
- 出险时间/地点/环节：
- 损失情况简述：
- 预估金额：
- 已有材料：照片、运单、交接…
- 联系人：
```

## 5. 催保司定损

```
您好，案件 {案号/运单} 已于 {日期} 报案，目前待定损已 {N} 天。
我们可补充：{维修报价/发票/现场配合时间}。
请告知预计定损时间与还缺材料清单，便于一次补齐。谢谢。
```

## 6. 内部升级（老板）

```
【理赔升级】
运单… 诉求 ¥… 类型…
证据完备度：… 初判置信度：…
风险：客户升级/超保额/责任不清
请拍板：① 报案 ② 协商上限 ③ 法务接入
```

## 禁止

- 威胁、侮辱、教唆作假  
- 未授权承诺「全额必赔」「一定赢」
