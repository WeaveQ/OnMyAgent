# 分类型证据清单

每项标记：`已有` / `缺失` / `薄弱` / `不适用`。

## 通用（所有案件）

| 证据 | 说明 |
|------|------|
| 运单/托运约定 | 线路、时效、保价、包装约定 |
| 货值证明 | 发票、合同、报价单 |
| 时间线节点 | 提送装载到达签收时间 |
| 各方书面沟通 | 微信/邮件，注意保全 |

## 外包装破损 / 内物损坏

| 证据 | 说明 |
|------|------|
| 卸货现场全景+特写 | 箱体六面、破口、内物、标签运单同框 |
| 装车前照片（若有） | 对比用 |
| 交接单/签收单 | 是否批注异常 |
| 司机/装卸说明 | 书面简要经过 |
| 包装是否达标照片 | 原厂包装、填充、木架等 |
| 监控（装/卸区） | 时间点 |

## 水湿

| 证据 | 说明 |
|------|------|
| 水渍范围与高度 | 多角度 |
| 车厢/雨布/仓储环境 | 渗漏线索 |
| 天气与时间 | 降雨节点（可后补公开天气） |
| 是否混装液体 | 邻货情况 |

## 延误客诉

| 证据 | 说明 |
|------|------|
| 约定时效条款 | 合同/运单/聊天确认 |
| 实际轨迹与到车时间 | GPS/到车登记 |
| 延误原因说明 | 天气、查扣、压车、等通知 |
| 客户损失主张依据 | 停线证明等（有则收，审慎采信） |

## 丢件 / 货差

| 证据 | 说明 |
|------|------|
| 装车交接件数 | 网点→司机 |
| 卸车交接件数 | 司机→网点/客户 |
| 监控装车/卸车 | 关键 |
| 在途是否甩挂换车 | 节点说明 |
| 库存台账 | 是否仍在库虚账 |

## 拍照要点（给现场）

1. 先全景后特写；含运单号/货签。  
2. 破损处放参照物（尺/硬币）。  
3. 原包装与内填充是否完好分别拍。  
4. 尽量在卸货当下、移动货物前拍。  

## 材料包目录模板

```text
CLAIM-{waybill}/
  01-运单与约定/
  02-照片与视频/
  03-交接与签收/
  04-说明与笔录/
  05-货值证明/
  06-保险与沟通记录/
  07-进度与结论/
```
