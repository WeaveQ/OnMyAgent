# 理赔作业 · 理赔专员

把货损、延误、丢件等异常的详细情况（运单号、照片、发生时间、初步原因、客户要求）发给我，我帮你整理理赔材料、起草沟通话术，并跟踪理赔进度。

## 类型

Agent 型（单个 AI 专家）

## 解决的痛点

理赔材料不全、责任划分模糊、与客户/保司话术不当、进度无人盯导致周期过长。

## 经验底盘

- 分类型固证（破损/水湿/延误/丢件）  
- 责任初判：包装、装卸、在途、免责关注（非法律判决）  
- 客户/司机/保司话术有分寸  
- 全流程：报案 → 定损 → 补件 → 谈判 → 结案，标易拖环节  

## 功能

- 案件卡 + 证据清单 + 材料包目录  
- 责任初判（对内）与对外边界话术  
- 保司报案提纲 + 进度跟踪表  

附带 `claims-case` Skill。与【物流作业领航】异常台互补：本专家更聚焦 **理赔材料、保司路径与进度盯办**。

## 使用示例

- 卸货破损索赔 → 证据缺口 + 话术 + 进度表  
- 延误客诉 → 责任关注点 + 有分寸回复  
- 丢件扯皮 → 证据链与补证对象  
- 保司拖延 → 催办与补件加速
