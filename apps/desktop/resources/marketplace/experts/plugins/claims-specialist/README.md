# 理赔作业 · 理赔专员

把货损、水湿、延误、丢件的事实与证据索引发给我，我会按异常类型跑固定证据清单，输出带条件与缺口的责任草稿、客户/保司双话术和可追踪进度表。

## 类型

Agent 型（单个 AI 专家）

## 解决的痛点

理赔材料不全、责任划分模糊、与客户/保司话术不当、进度无人盯导致周期过长。

## 经验底盘

- 分类型固证（破损/水湿/延误/丢件）  
- 责任初判：包装、装卸、在途、免责关注（非法律判决）  
- 客户/司机/保司话术有分寸  
- 全流程：报案 → 定损 → 补件 → 谈判 → 结案，标易拖环节  

## 功能

- 案件卡 + 证据清单 + 材料包目录  
- 责任初判（对内）与对外边界话术  
- 保司报案提纲 + 进度跟踪表  
- `claim-case.json` → `.process` 看板 → 材料包/双话术/进度 CSV 的确定性 preview/export

附带 `claims-case` Skill 与 `build_claim_artifacts.py`。只生成作业草稿，不自动认责、报案、承诺赔付或外发。

## 使用示例

- 卸货破损索赔 → 证据缺口 + 话术 + 进度表  
- 延误客诉 → 责任关注点 + 有分寸回复  
- 丢件扯皮 → 证据链与补证对象  
- 保司拖延 → 催办与补件加速
