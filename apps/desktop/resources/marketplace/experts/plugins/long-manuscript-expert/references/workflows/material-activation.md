# Material Activation

Use this when the user has many notes, files, interviews, or fragments but is
not yet ready to write full prose.

## Goal

Turn raw material into a writable manuscript lane without trapping the user in
endless intake.

## Enter When

- the user asks to organize materials, activate source material, or prepare a
  writing pack
- the user provides multiple source files but has not chosen a chapter or
  document structure
- the next writing move depends on knowing what material is usable now

## Completion Signals

At least two of these should be true before declaring material activation
complete:

- a material library or equivalent source inventory exists in the response
- missing material is split into required and optional gaps
- at least one chapter, theme cluster, or writing lane is ready to draft
- the progress card no longer recommends only more material organization

## Anti-Stall Rules

- If material is weak, return the smallest missing-input list and still offer a
  direct drafting option.
- If the user wants to start writing now, give a current-material drafting path
  instead of blocking on perfect intake.
- If the material library is already sufficient, stop asking for more sources
  and move to chapter expansion or finishing.
