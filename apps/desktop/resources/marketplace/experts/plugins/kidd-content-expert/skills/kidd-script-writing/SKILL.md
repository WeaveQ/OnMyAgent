# 基德短视频文案创作 Skill

基于"吟游诗人基德"视频风格，创作深度分析类短视频文案。

## 通用特征（所有题材）

- 口语化表达：短句碎拍、北方方言、网络热梗
- 五段式结构：Hook → 知识铺垫 → 核心问题 → 反常识解答 → 连锁收尾
- 农夫小姐机制：预设观众代言人"农夫小姐"，创造对话感
- 数据可视化：善用表格呈现复杂信息，表格后必须有口语化总结
- 反常识结论：推翻观众第一直觉，提供深度洞察

## 宇宙科普专属特征

- 科幻感开头：使用公式、数字、特效描述制造冲击力
- 时间/空间尺度感：善用"40亿年"、"天文单位"等宏大概念
- 硬核数据+幽默表达：将复杂科学数据用幽默方式呈现
- 步骤化叙事：将复杂过程分解为清晰阶段（第一步、第二步...）
- 视觉化提示：标注B-roll、动画、特效需求
- 论文引用习惯：引用科学论文和DOI增强权威性

## 适用题材

- 国际政治、财经分析、地缘冲突、社会热点解读
- 宇宙科普、天文物理、行星科学、太空探索
- 科技前沿、硬核科普、工程奇迹

## 执行步骤

按以下7步严格创作：

### Step 1: 主题分析与信息收集

1. 识别主题中的【热点事件】和【矛盾冲突】
2. 思考：观众的第一直觉是什么？（需要推翻的预设）
3. 思考：反常识的真相是什么？（核心洞察）
4. 列出需要用到的【数据/事实】和【地理位置/时间线】
5. 宇宙科普题材额外检查：是否需要引用科学论文？是否涉及时间/空间尺度？是否需要公式或数字开头？

输出：核心冲突点、观众预设、反常识结论、需查证的数据点、适用题材类型

### Step 2: 开头段（Hook）创作 — 黄金3秒

**时政/财经公式：**
```
[热点事件，1句] + [反常/矛盾，1-2句] + [农夫小姐反应，1句] + [夸张比喻，1句] + [价值承诺，1句]
```

**宇宙科普公式：**
```
[公式/数字/特效描述，1句] + [反常识对比，1-2句] + [时间/空间尺度感，1句] + [农夫小姐反应，1句] + [价值承诺，1句]
```

语言要求：
- 开头第一句必须是冲突/反常/科幻感
  - ❌ "今天我们来聊..."
  - ✅ "美伊谈崩了..."（时政）
  - ✅ "毁灭火星很简单。根据重力结合能公式..."（科普）
- 立即引入"农夫小姐"："这时农夫小姐就懵逼了..."
- 使用夸张类比：「严肃议题」跟「荒诞场景」有什么区别？
- 结尾承诺："言归正传，这期视频用几分钟时间，带大家看懂[主题]，全是干货。"

### Step 3: 知识铺垫段创作

**通用公式：**
```
[地图/背景引入] → [对比：常见误解 vs 真相] → [数据冲击] → [法律/技术解释] → [农夫小姐提问] → [深化解释]
```

**宇宙科普公式：**
```
[科学背景引入] → [对比：常见误解 vs 真相] → [时间/空间尺度冲击] → [科学原理解释] → [农夫小姐提问] → [深化解释 + 论文引用]
```

技巧：
1. 破除误解：用"不是...是..."结构
2. 数据冲击：使用倍数关系（5倍、80%）
3. 农夫小姐提问：预设观众疑问
4. 时间尺度感："40亿年前的太阳系到底发生了什么？"
5. 空间尺度感："全部挤在1个天文单位以内"
6. 论文引用："2017年的一篇论文认为...（论文链接：https://doi.org/...）"
7. 硬核数据+幽默："相当于两个地球现有核能工业总和"

### Step 4: 核心问题段创作

**通用公式：**
```
[问题提出] → [枚举展示，排比句式] → [数据表格] → [逻辑推演] → [结论]
```

**宇宙科普公式：**
```
[问题提出] → [步骤化叙事：第一步/第二步...] → [数据表格/对比图] → [科学原理推演] → [结论 + 视觉化提示]
```

技巧：
1. 过渡句式："好那现在讲第二个问题，[问题]？我看全网竟然没人具体讲过。"
2. 枚举排比："那我做了个地图，这是...这是...这是..."
3. 数据表格：必须包含，表格后用口语总结："瞎子都能看出来..."
4. 步骤化叙事："改造火星的第一步，不是种树，也不是造房子，而是——找回丢失的火星磁场护盾"
5. 视觉化提示："（B-roll：快闪俯瞰镜头——巨大的轨道反光镜、地表燃烧的工厂）"

### Step 5: 反常识解答段创作

```
[推翻预设：下意识觉得...但其实...] → [数据支撑] → [分层分析] → [案例佐证] → [强调客观]
```

技巧：
1. 推翻预设："农夫小姐们下意识会觉得欧美最急，但其实..."
2. 分层论证：第一层（首先...）→ 第二层（其次...）→ 第三层（再加上...）
3. 案例佐证：具体公司/数字/时间点
4. 强调客观性："说实话最急的是日韩，任我再怎么查资料都是这个结果，纯客观。"
5. 科学悬念："难道，这不是月球灾难，而是太阳系灾难？"
6. 多假说并列："还真有一种观点认为，可能是有的。"
7. 科学不确定性："科学家甚至有50%的把握说..."

### Step 6: 连锁收尾段创作

```
[反转词：反转了] → [第一层逻辑] → [第二层逻辑] → [第三层逻辑] → [恶性循环总结] → [金句收尾]
```

技巧：
1. 连锁连接词："反转了，其实..."、"而且，你要知道..."、"最后一步..."
2. 情绪递进：直接逻辑 → 连锁反应 → 系统性影响
3. 金句收尾："大家日子都不好过，所以这事办的你说，[总结]的话，真[结论]了。"
4. 哲学升华（宇宙科普）："一切刚刚好，简直是宇宙对我们的馈赠"
5. 科学名言引用："早在1911年，火箭之父康斯坦丁·齐奥尔科夫斯基就写下了那句预言"

### Step 7: 整体质检与润色

| 检查项 | 标准 |
|---|---|
| 开头3秒 | 有冲突/反常/科幻感 |
| 农夫小姐 | ≥2次出现 |
| 夸张比喻 | ≥1个 |
| 数据表格 | ≥1个 |
| 反常识点 | ≥1个 |
| 口语程度 | 读起来像说话 |
| 连锁收尾 | 3层以上逻辑 |
| 金句结尾 | 有 |
| 论文引用（宇宙科普） | 有科学来源 |
| 视觉化提示（宇宙科普） | 有B-roll/动画标注 |
| 尺度感（宇宙科普） | 时间/空间宏大概念 |

润色要点：替换学术术语为口语、长句拆短句(≤15字/句)、增加语气词（搁这、架不住、真没辙、最骚的是）、加入反问句、平衡硬核科学与幽默

| 科学引用 | 论文、DOI | "2017年的一篇论文认为..." |

## 风格禁忌

| 禁忌 | 修改方向 | 错误示例 | 正确示例 |
|---|---|---|---|
| 学术术语 | 口语解释 | "地缘政治博弈" | "两个大爹没给别人留一口气" |
| 长从句 | 短句拆分 | "鉴于当前形势的复杂性..." | "你知道...吗？" |
| 中性立场 | 鲜明观点 | "各方都有自己的考虑" | "专业掐脖子" |
| 被动语态 | 主动语态 | "石油被送往..." | "80%的油往亚洲送" |
| 抽象概念 | 具体场景 | "产生负面影响" | "裂解装置开工率暴跌至50%" |
| 纯硬核无幽默 | 硬核+幽默 | "根据公式U=3GM²/5R" | "你只需要把核弹绑一起乘以一亿倍" |
| 无视觉化提示 | 添加标注 | "火星表面温度变化" | "（B-roll：快闪俯瞰镜头——...）" |
| 无论文引用 | 添加来源 | "科学家发现..." | "2017年论文认为...（链接）" |
