# 窗口期 / Alpha 意识方法论

> **何时加载**：用户问题包含 风口 / 红利 / 现在火 / 新词 / Alpha / 窗口期 / 抢机会

**每次出现新词，都是财富重新分配的大机会。**

红利 = 供需失衡最锐利的窗口期，一般 3-4 个月。

---

## Alpha 三层

不是所有人都能抓窗口。差距在 3 层：

### 1. 信息差（起跑线）

谁先听到 = 起跑线提前。

- 关注头部信息源（Hacker News / Twitter / Product Hunt / 即刻）
- 订阅高质量 newsletter（每月 3-5 个，看完淘汰）
- 加几个高质量社群（生财有术 / 行业群 / Discord）
- **每周问自己**：本周我看到的最新的 3 个词是什么？

### 2. 行动速度（十倍差）

听到了 ≠ 做了。听到 → 做出来 → 上线，时间差就是 alpha。

- 看到机会到上线 MVP 的时间
- 一周 vs 三个月——同一个机会，结果完全不同
- 一人公司的最大优势：决策速度（不开会，不汇报）

### 3. 判断力（过滤 90% 噪音）

不是每个新词都是机会。要过滤：

- 这是真需求还是炒作？
- 这是窗口期还是泡沫？
- 这事我能做吗？还是看着热闹？

> **判断力不是知识，是**——见过 100 个机会被毁的方式。

## 新词站判断 4 问

不是所有新词都值得做"新词站"。问 4 个问题：

### Q1：搜索斜率明显上升？

工具：Google Trends / 百度指数 / Semrush / Ahrefs

看的不是绝对值，是**斜率**。从 0 → 100% 的增长比从 100% → 200% 重要。

- 真窗口：3 个月内从 1k/月搜索量涨到 50k/月
- 假风口：搜索量持平或缓慢增长

### Q2：用户想"用"还是想"看"？

- ✅ 想"用"：搜索词后面跟着 "tool" / "online" / "free" / "vs" / "alternative" → 高意图，易变现
- ❌ 想"看"：搜索词后面跟着 "what is" / "explained" / "tutorial" → 信息消费，难变现

例子：
- "GPT-4 vs Claude" → 想用（决策中）
- "what is GPT-4" → 想看（学习中）

### Q3：官方供给是否贵 / 难 / 慢？

新词出现初期，官方往往：
- 贵（API 价格高）
- 难（开发门槛高）
- 慢（功能不完整）

这就是窗口——**你做一层 wrapper / 工具层 / 入口卡位**就有市场。

### Q4：48 小时能做出 MVP 吗？

如果不能，窗口可能已经关闭。一人公司抓窗口的核心优势是 48-72h MVP。

> 超过一周，要么是机会假，要么是你切口太大。

## 新词站不是什么

不是：

- ❌ 资讯站（写文章解释新概念）
- ❌ 蹭热点的内容站
- ❌ 长文 SEO 站

是：

- ✅ **入口卡位**——用户已经在搜，你接住高意图流量
- ✅ **工具层**——把官方 API 包装成更易用的工具
- ✅ **代理层 / 转售层**——用户不想直接对接 API
- ✅ **服务层**——提供基于新概念的 done-for-you 服务

---

## 适合一人公司的窗口

**适合**：

- 单点工具（一个明确功能 + 一个明确价格）
- 信息聚合（多个数据源 → 一个仪表盘）
- 模板 / 资源站（变现 = 一次性付费）
- 教程 + 工具组合（教用户用某个新东西 + 卖工具）

**不适合**：

- 平台型（需要双边市场）
- 重运营（需要内容团队 / 客服团队）
- 强 SEO 长期投入（一人公司没耐心）
- 需要快速融资才能跑通的（说明赛道太重）

---

## 危险信号

- "我在等 GPT-5 出来再做" → 真正抓 alpha 的人在 GPT-3 时已经在做
- "等市场成熟再进" → 成熟 = 红海 = 没你的份
- "我不想抢热点" → 你不是不想，是没准备好
- "听说 X 很火，我也想做" → 听说 ≠ 看到。看 Google Trends 真实数据
- "等我学会再做" → 边学边做才是 Alpha 选手

## 收尾必做

让用户**今晚**就做：

1. 列出本周看到的 3 个新词
2. 用 Google Trends 看 3 个词的曲线
3. 选 1 个"上升斜率最陡 + 想用而非想看 + 官方供给慢"的词
4. 48 小时内做出 v0（哪怕是个 landing page + Notion 后端）

> **抓 alpha 不是聪明，是动手快。**
