# 常见货物描述与特殊要求表达对照

## 一、口语化货物描述 → 规范表述

| 客户口语 | 规范品名 | 备注 |
|---------|---------|------|
| 钢材、钢筋、型材 | 钢材（具体：螺纹钢/钢板/型材） | 追问具体品类与单件重量，涉及吊装需求 |
| 机器、设备 | 机械设备（具体：注塑机/机床/电机） | 追问尺寸重量，判断是否超限 |
| 家具 | 家具（板式/实木/沙发） | 易损，提示包装与送货上楼需求 |
| 布料、布匹 | 纺织品（布匹） | 防潮要求 |
| 食品 | 食品（预包装/生鲜） | 生鲜涉及冷链温控，见特殊要求 |
| 酒水 | 酒类饮品 | 易碎 + 部分属危险品（高度白酒限运） |
| 电池 | 电池（锂电/铅酸） | 锂电池属第 9 类危险品，需提示资质 |
| 油漆、涂料 | 涂料/油漆 | 多为危险品（易燃），提示资质或拒收 |
| 化肥、农资 | 化肥/农资 | 部分有腐蚀性，注意包装 |
| 配件 | 具体品类配件（汽配/机械配件） | 追问是否急件，常配时效要求 |
| 电商货、快递件 | 电商包裹（混装） | 按件数 + 总体积记录 |

## 二、时效表达转换

| 客户口语 | 规范时效档位 |
|---------|------------|
| 越快越好、很急、马上 | 加急（追问期望发货日期与到达时限） |
| 明天到、后天要用 | 次日达 / 隔日达（按日期确认） |
| 不急、什么时候都行 | 普通时效 |
| 今天必须发走 | 当日发运（确认截单时间） |
| 等通知 | 等通知发货（标注待触发，列入台账跟进） |

## 三、特殊要求表达与风险提示

| 类型 | 客户常见说法 | 规范记录 | 必须提示 |
|------|------------|---------|---------|
| 尾板车 | "要尾板""没叉车" | 车型要求：带尾板车辆 | 确认装卸两端都无叉车还是仅一端 |
| 冷链 | "要冷藏""怕化" | 冷链运输（追问温控区间） | 温控范围（冷藏 0-4℃/冷冻 -18℃）需明确 |
| 易碎 | "怕压""轻拿轻放" | 易碎品（提示包装加固） | 建议打木架/木箱，破损责任约定 |
| 回单 | "要回单""签单返回" | 签收回单服务 | 回单返回方式与时限 |
| 等通知放货 | "先别送""等我电话" | 等通知放货 | 仓储费与免租期提示 |
| 大件/超限 | "机器有点大" | 超限货物（追问长宽高） | 超宽超高需确认路线与办证 |
| 危化品 | "油漆""酒精""电池" | 危险品（按类别） | **必须具备危化品承运资质**，普货车辆拒运 |
| 高价值 | "这货很贵" | 高价值货物 | 建议保价，追问声明价值 |
| 送货上楼 | "要搬上楼" | 送货上门含上楼 | 是否有电梯、楼层数 |

## 四、禁运/限运提醒词库

油漆、稀释剂、酒精、液化气罐、锂电池（纯电池）、烟花爆竹、农药 → 触发危险品提示流程：先确认货物品名与类别，再提示资质要求，不确定时建议客户提供 MSDS。
