# 物流单据类型与通用格式

## 选择原则

用户模板 > 承运商/系统模板 > 本技能通用格式。“通用格式”是为日常普通货运整理的业务版式，不是法定统一样式。

| 单据类型 | 主要用途 | 通用格式重点 |
| --- | --- | --- |
| 物流单 | 记录一票普通货物的托运、承运、收货和结算信息 | 单号/日期、托运方、收货方、起讫地、货物、交接、运输/结算、签收 |
| 发货单 | 发货方/仓库向收货方交付货物的明细凭证 | 发货单号、订单号、发货仓、收货方、品名/规格/单位/应发/实发、批次、经手/验收 |
| 发车单/派车单 | 调度下达车辆和司机的运输任务 | 任务号、车牌/车型、司机/电话、计划装卸时间、路线、货物、装卸要求、调度/司机确认 |
| 运单 | 作为承运过程和运输合同/证明的业务凭证 | 运单号、托运人、承运人、收货人、货物、起讫地/区段、运费条款、交接、声明与签章 |

## 公开依据

- 国家标准全文公开系统显示 [GB/T 33449-2016《物流单证基本要求》](https://openstd.samr.gov.cn/bzgk/gb/newGbInfo?hcno=FF012ACBFF59194175B063E27CB60E6F) 已废止，只能作历史参考，不得宣称为现行标准。
- 交通运输部发布的 [JT/T 931.1—2014《国际道路运输电子单证格式 第1部分：货物运单》](https://xxgk.mot.gov.cn/2020/jigou/kjs/202006/t20200623_3316937.html) 明确货物运单包含数据项属性、信息模型和信息表。
- 交通运输部公开的集装箱多式联运运单资料将共性内容归纳为多式联运经营人、托运人、货物信息、区段承运人和收货人等部分，可作通用运单分区参考：[编制说明](https://big5.mot.gov.cn/gate/big5/xxgk.mot.gov.cn/2020/jigou/kjs/202307/P020230728387872654752.pdf)。
- 快递面单是单独场景，使用现行 [GB/T 41833-2022《快递电子运单》](https://openstd.samr.gov.cn/bzgk/std/newGbInfo?hcno=A95B161B8D0E3D589ACF02355DA18DFB)，不要将快递面单版式直接套用到普通物流托运单。

## 专项场景

- 危险货物、冷链、超限运输、国际道路运输、多式联运等场景有额外的法规、资质和字段要求。
- 遇到专项场景时，先提示用户按承运商或主管部门模板办理；通用单据不代替专用单据。
