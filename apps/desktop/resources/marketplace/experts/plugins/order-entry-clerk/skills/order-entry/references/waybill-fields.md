# 物流运单标准字段

## 必填字段（缺失必须追问）

| 中文字段 | JSON 字段名 | 说明 | 格式校验 |
|---------|------------|------|---------|
| 发货人姓名/公司 | shipper.name | 发货方联系人或公司名 | 非空 |
| 发货人电话 | shipper.phone | 联系电话 | 手机号 11 位或带区号座机 |
| 发货地址 | shipper.address | 装货地址 | 至少细化到区/镇级，有具体门牌或地标更佳 |
| 收货人姓名/公司 | consignee.name | 收货方联系人或公司名 | 非空 |
| 收货人电话 | consignee.phone | 联系电话 | 手机号 11 位或带区号座机 |
| 收货地址 | consignee.address | 卸货地址 | 至少细化到区/镇级 |
| 货物名称 | cargo.name | 品名，需规范表述 | 非空，禁用"东西""货"等模糊词 |
| 数量/件数 | cargo.quantity | 件数或台数 | 数字 + 单位 |
| 重量或体积 | cargo.weight / cargo.volume | 计价与配载依据，至少一项 | 重量 kg/吨；体积 立方米/方 |
| 时效要求 | timeline | 期望发货/到达时间 | 明确时间点或档位（当日达/次日达/普通/加急） |

## 选填字段（缺失可后补，但建议提示）

| 中文字段 | JSON 字段名 | 说明 |
|---------|------------|------|
| 车型要求 | vehicleRequirement | 如 4.2 米厢车、7.6 米、9.6 米、尾板车、高栏、平板、冷藏车 |
| 特殊要求 | specialRequirements | 易碎、防潮、冷链温控、回单、送货上门、卸货协助、等通知放货等 |
| 货物价值 | cargo.declaredValue | 涉及保价，高价值货物建议提示保价 |
| 包装方式 | cargo.packaging | 木箱、托盘、纸箱、裸装等 |
| 结算方式 | paymentMethod | 现付、到付、月结、回单付 |
| 是否保价 | insurance | 保价金额 |
| 备注 | remarks | 其他补充信息 |

## 字段置信度标注规则

- **高**：原文明确给出，格式合规
- **中**：原文有但表述模糊（如"大概两吨""明天左右"），需规范转换
- **低**：依赖识别（语音/手写）且结果存疑，或根据历史偏好补全，必须请客户确认

## 常见校验问题

- 手机号少位/多位 → 标红追问原号码
- 地址只有城市名（如"发广州"）→ 追问区/镇 + 详细地址
- 货物只有品名没有数量重量 → 无法计价配载，必须追问
- "越快越好""尽快" → 转换为"加急"，并追问可接受的时效档位与期望日期
