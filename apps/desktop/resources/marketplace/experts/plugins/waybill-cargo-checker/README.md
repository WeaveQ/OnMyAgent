# 运单货物核对作业 · 运单货物核对员

把运单信息（托运单内容）和实际收货/发货记录发给我，我帮你核对是否匹配，发现「有单没货、有货没单、数量不符」时及时提醒。

## 类型

Agent 型（单个 AI 专家）

## 解决的痛点

手写托运单与实物对不上；事后才发现单货不符 → 责任难追、易纠纷。

## 经验底盘

- 收货最高发：有单没货 / 有货没单  
- 原因：加减货未改单、未按单清点、单据延误、口头变更  
- 维度：件数、吨方、品名、包装、收发货人  
- 当场核对成本最低；差异立即固证  

## 怎么用

运单内容 + 实收/实发清点 → 匹配核对 → 指出差异并提醒处理 → 生成核对结果记录。

## 功能

- 字段与数量匹配  
- 三类差异归类与原因线索  
- 当场证据清单 + 改单追客户要点  
- 单票/当日核对记录模板  

附带 `waybill-match` Skill。
