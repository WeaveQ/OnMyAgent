## 匹配优先级

1. 运单号  
2. 发货人 + 品名 + 件数（弱）  
3. 仍对不上 → 分别进「有单无货匹配」「有货无单」池，禁止硬凑  

## 字段比对规则

| 字段 | 一致条件 | 备注 |
|------|----------|------|
| 件数 | 相等 | 主硬条件 |
| 重量 | 差在容差内（默认 3% 或用户定）或一侧缺失 | 缺失标待补 |
| 体积 | 同上 | 泡货敏感 |
| 品名 | 同义可接受须注明 | 「配件」vs 具体名 |
| 包装 | 类别一致 | 纸箱/木箱混用标差异 |
| 收发货人 | 关键名一致 | 电话可另核 |
