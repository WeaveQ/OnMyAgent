# 差异类型与原因线索

## 类型码

| 码 | 中文 | 定义 |
|----|------|------|
| MATCH | 一致 | 关键字段在规则内一致 |
| WAYBILL_NO_CARGO | 有单没货 | 有运单，现场找不到对应货（或少到 0） |
| CARGO_NO_WAYBILL | 有货没单 | 有货无有效运单/托运单 |
| QTY_SHORT | 数量短少 | 实收/实发 < 单载件数 |
| QTY_OVER | 数量溢余 | 实收/实发 > 单载件数 |
| WEIGHT_VOL_GAP | 吨方不符 | 超容差 |
| NAME_PACK_GAP | 品名/包装不符 | 货不对板 |
| PARTY_GAP | 收发货人信息不符 | |
| SUSPECT_OCR | 识别存疑 | 手写/照片不清，先复核 |

可多码并存（如短少 + 品名存疑）。

## 常见原因线索（非定责）

| 现象 | 可能原因 |
|------|----------|
| 有单没货 | 货未到、卸错区、已先发未销单、单货分离 |
| 有货没单 | 临时加货未出单、单据延误、口头加货 |
| 短少 | 客户少发、途损、漏卸、清点漏数 |
| 溢余 | 客户多发、两票混装、错装 |
| 吨方差 | 报称不准、泡重、过磅差异 |
| 品名差 | 笼统品名、错贴唛头 |

## 处理优先级

1. 有货没单 / 有单没货（阻断混入正常库）  
2. 短少/溢余  
3. 吨方与品名  
4. 仅 OCR 存疑  

## 建议动作（作业）

| 类型 | 立即动作 |
|------|----------|
| 有单没货 | 搜货位/车次；联系发货方；暂缓签「货已齐」 |
| 有货没单 | 隔离堆放；拍照；追补托运单或拒收政策按公司规 |
| 短少/溢余 | 复盘；双方确认差量；改单或批注 |
| 品名/包装 | 开箱规范（授权下）；改单品名 |
