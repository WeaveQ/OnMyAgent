# 附加费防漏检查清单

逐项标记：`含在总价` / `另计` / `不适用` / `待确认`。未确认项不得默认「免费」。

| 类别 | 检查项 | 常见漏报场景 |
|------|--------|--------------|
| 干线 | 专线/整车/零担主体 | 只报干线忘提送 |
| 提货 | 提货费、多点提、等装 | 「顺便提一下」未计价 |
| 送货 | 送货费、多点送、分卸 | 园区多仓 |
| 装卸 | 装车/卸车/叉车工 | 口头「你们负责卸」未计价 |
| 税费 | 含税与否、专票成本 | 报未税当含税 |
| 末端 | 上楼、进仓、预约进场 | 电商仓/超市 |
| 车型能力 | 尾板、飞翼、低栏、冷藏 | 客户口头「要尾板」 |
| 等待 | 等通知放货、压车超时 | 卸货慢 |
| 回单 | 纸质回单费、加急回单 | 月结要原件 |
| 保险 | 保价/保险费 | 高货值 |
| 路桥油 | 过路费、油补（若约定） | 包车口径 |
| 放空 | 放空风险、空驶补贴 | 客户可能取消 |
| 异形 | 超长超高超重附加 | 设备件 |
| 时段 | 夜班、周末、节假日 | 明天急提 |
| 渠道 | 信息费（对内可单列） | 对客总价口径 |

## 输出形式

```markdown
### 附加费检查
| 项 | 状态 | 备注 |
|----|------|------|
| 提货 | 含在总价 / 另计 ¥… / 待确认 | |
| 送货 | | |
| 尾板 | | |
| … | | |
```

## 规则

- 客户说「包干」时，仍要在内部列清包干覆盖范围，避免业务漏项。  
- 从「最快」改到「最便宜」时，写明 **砍掉或降级了哪些服务**，而不是只改一个总价数字。
