# 成本构成、隐藏条件与历史价沉淀

## 基础运价成本构成（对内检查）

测算成本区间或底价时，按需勾选是否已覆盖（有数填数，无数写「待填」）：

| 构成 | 说明 | 备注 |
|------|------|------|
| 油费 | 里程 × 油耗口径或包干油 | 油价波动大时注明假设日 |
| 过路费 | 高速/路桥 | 整车包车更敏感 |
| 车辆折旧 / 轮胎损耗 | 按车型分摊 | 可并入「车天成本」 |
| 司机成本分摊 | 工资+食宿摊到车天/票 | |
| 配载损耗 / 放空缓冲 | 配货空仓、等货 | 薄利线要留缓冲 |
| 网点/操作 | 专线中转、装卸班组 | |
| 保险与合规 | 车险/货运险分摊 | 与对客保价分开 |
| 税费口径 | 含税/未税 | **必须写清** |

> 无用户成本表时：只输出上表结构 + `示意` 比例关系，禁止假装精确全网价。

## 车型与配载（快速口径）

| 场景 | 常见车型 | 计费倾向 |
|------|----------|----------|
| 同城/短驳 | 4.2 厢等 | 常按车次/半天 |
| 区域整车 | 6.8 / 9.6 / 13 | 包车或车公里 |
| 专线配货 | 拼载 | 吨/方档 + 提送 |
| 急件 | 整车或优质专线 | 时效溢价进「最快」档 |

整车与配货 **单价不可混用**；从「最快」改「最便宜」时优先改配载方式/时效，而不是无说明狂砍。

## 隐藏条件清单（询价必扫）

| 条件 | 问法/识别 | 对报价影响 |
|------|-----------|------------|
| 是否含税 | 「含税价吗」「专票」 | 总价与开票成本 |
| 回单形式 | 电子 / 原件 / 加急回单 | 回单费、账期、结账卡点 |
| 装卸责任 | 含装含卸 / 仅运输 / 需叉车工 | 装卸费、等待 |
| 旺季/节假附加 | 双11、春节前后、政策限行 | 附加或拒单风险 |
| 等通知放货 | 到站等指令 | 仓储/压车规则 |
| 进仓预约 / 月台 | 电商仓、商超 | 预约费、超时 |
| 可叠压 / 异形 | 设备件、怕压货 | 配载率、车型 |
| 运力松紧 | 用户告知或当日调度体感 | 砍价空间、是否加急溢价 |

输出中单列：**已确认 / 待确认 / 假设**，避免静默默认「全包」。

## 易漏附加费（实战高频）

优先检查：提送、装卸、尾板、进仓预约、回单原件、等通知、多点分卸、夜班/周末、旺季附加、信息费口径（对内）。

## 薄利与策略提醒（有依据才写死）

- 用户或历史数据表明某线「价格战/利润薄」→ 在平衡/最便宜档标注 **薄利提醒** 与不可再砍红线。  
- 无数据 → 用中性表述：「该线竞争充分时注意留附加费与放空缓冲」。

## 历史价格沉淀（前后一致）

每票报价草稿可附：

```text
【本票价格备忘】
- 采用指导价/历史成交：{有则填，无则「未提供」}
- 计费口径：{吨/方/车/抛重比}
- 含税：{是/否/待确认}
- 三档对客锚点：最快 ¥… / 平衡 ¥… / 最便宜 ¥…
- 底价红线：¥… 或待填成本
```

便于同一客户再次询价时对齐口径，缓解「全靠个人经验、无法沉淀」。
