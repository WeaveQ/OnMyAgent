# 报价与砍价话术

## 1. 首次报价（清晰不含糊）

```
{称呼}，{线路} {吨/方}，时效{…}，我给您三个方案方便选：

1）最快：{方式/时效}，合计约 {金额}，含{…}，不含{…}
2）平衡：{…}，合计约 {金额}（多数客户选这个）
3）省心省钱：{…}，合计约 {金额}，需{放宽时效/自提等}

说明：{含税与否/回单形式/装卸是否含}。有效期 {如24小时}，以实际吨方和提送地址确认为准。
您更看重时效还是总价？
```

## 2. 只推一档（客户嫌多）

```
按您要的{时效/预算}，建议走【平衡/最快/最便宜】：合计 {金额}，含{…}。
不含{…}。需要我按这个锁一版给您书面确认吗？
```

## 3. 客户嫌贵

```
理解预算。当前这版已含{提送/时效/装卸}；若去掉{某服务}或放宽到{时效}，可落到【最便宜】大约 {区间}。
您是想保时效，还是优先压总价？我按您的优先级改一版，不瞎砍服务硬降价。
```

## 4. 客户比价 /「别人更低」

```
可以对比。请看是否含{提送/装卸/进仓/回单/尾板/保险}、是否含税、时效是否{…}。
若对方是裸干线价，把同样服务对齐后我再给您一版对等口径；该减的项我明示减，不藏费用。
```

## 5. 要死价 / 逼到底

```
死价可以，需确认吨方、地址、含税和是否含{…}。
含全项的死价是 {金额}；若接受{自提/普快/电子回单}，死价可到 {金额}。
（对内：不低于底价 {底价}。）
```

## 6. 运力紧张时（有依据才用）

```
这几天{线路/车型}车比较紧，急走建议【最快】{金额}，位置有限。
若能{晚一天/配货}，【平衡/最便宜】仍可谈到约 {区间}。您更急还是更看预算？
```

## 7. 老客户弹性（有合作价值依据时）

```
您是老客户，这票我按【平衡】给到 {金额}，已含{…}；再往下就要减{服务}或改{时效}了。
长期走量我们另谈月度指导价，单票再死磕会伤时效和服务。
```

## 8. 守底价（对内给老板/业务）

```
对客可谈，但不建议低于 {底价}：再低会吃掉 {油/路桥/司机分摊/提送} 缓冲。
可换：减服务项 / 拼载配货 / 改提货时间 / 换最便宜档；
结合：当前运力{松/紧}、该客户历史{价值/回款}、本线是否薄利。
不要纯砍总价、服务不变。
```

## 9. 附加费解释（防纠纷）

```
总价里已含{…}。以下情况会另计或需现场确认：{等通知/压车超时/二次进仓/旺季附加/…}。
回单按{电子/原件}，装卸责任是{…}。装车前我们再对一次清单，避免到货扯皮。
```
