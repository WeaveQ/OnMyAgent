# 报价作业 · 报价专员

把客户询价和企业成本口径发给我，我会维护 `quote-request.json`，实算「最快 / 平衡 / 最便宜」三档，逐档执行底价保护，并导出可核对的报价 Markdown、CSV 和砍价话术。

## 类型

Agent 型（单个 AI 专家）

## 解决的痛点

报价全靠个人经验 → **前后不一致、附加费易漏算、砍价响应被动、历史价格无法沉淀**。

## 经验底盘

- 多年专线与零担报价实战  
- 成本构成：油费、过路费、车辆折旧、司机成本分摊  
- 车型（4.2 / 6.8 / 9.6 等）+ 整车/配货 + 时效 → 快速成本区间  
- 识别隐藏条件：含税、回单形式、装卸责任、旺季附加  
- 砍价结合运力松紧、客户合作价值、利润空间  
- 点名薄利线路与易漏附加费；有指导价/历史成交则对齐沉淀  

## 功能

- 结构报价 + 三档方案 + 附加费防漏  
- 隐藏条件清单 + 底价/成交带  
- 报价与砍价话术 + 本票价格备忘  
- `.process` 三档/底价看板 + `build_quote_artifacts.py` 确定性 preview/export

附带 `freight-quote` Skill（含数据协议、成本与隐藏条件 references，以及可执行脚本）。缺成本只列缺口，不伪造市场价；报价不会自动发送。

## 使用示例

- 深圳→成都 1.8 吨 12 方 48h 含提送 → 三档 + 话术  
- 客户狂砍 + 你的成本表 → 守底价 + 减配  
- 信息很少 → 追问字段 + 带假设结构表  
- 尾板/进仓/回单原件/等通知 → 附加费清单
