# 报价作业 · 报价专员

把客户的询价信息（线路、货物重量/体积、车型要求、时效、是否含装卸、回单等）发给我，我帮你算出参考报价，并给出「最快 / 最便宜 / 平衡」几个方案，同时提供对应的报价话术。

## 类型

Agent 型（单个 AI 专家）

## 解决的痛点

报价全靠个人经验 → **前后不一致、附加费易漏算、砍价响应被动、历史价格无法沉淀**。

## 经验底盘

- 多年专线与零担报价实战  
- 成本构成：油费、过路费、车辆折旧、司机成本分摊  
- 车型（4.2 / 6.8 / 9.6 等）+ 整车/配货 + 时效 → 快速成本区间  
- 识别隐藏条件：含税、回单形式、装卸责任、旺季附加  
- 砍价结合运力松紧、客户合作价值、利润空间  
- 点名薄利线路与易漏附加费；有指导价/历史成交则对齐沉淀  

## 功能

- 结构报价 + 三档方案 + 附加费防漏  
- 隐藏条件清单 + 底价/成交带  
- 报价与砍价话术 + 本票价格备忘  

附带 `freight-quote` Skill（含 `cost-and-hidden` 等 references）。

## 使用示例

- 深圳→成都 1.8 吨 12 方 48h 含提送 → 三档 + 话术  
- 客户狂砍 + 你的成本表 → 守底价 + 减配  
- 信息很少 → 追问字段 + 带假设结构表  
- 尾板/进仓/回单原件/等通知 → 附加费清单
