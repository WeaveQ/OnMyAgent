# 催回单与催补资料话术

## 原则

1. **一次说清**：缺哪些票、要原件还是电子、最晚何时回、回哪里/给谁。  
2. **列表编号**：方便司机/对方回复「1、3 已回」。  
3. **语气**：客气、简短，适合微信；对承运商可略正式。  
4. **不威胁伪造**：可说明「影响结账进度」，不诱导造假签收。

## 1. 司机群（超期原件）

```
各位师傅，对账要用回单原件。以下运单已超过 {N} 天未交回，麻烦本周内交到{网点/调度}，或回复预计交回日期：
1）{运单号} {线路} {师傅}
2）…
收到回复「已交/日期」我好销账，谢谢！
```

## 2. 单个司机私聊

```
{师傅}，您跑的 {运单号}（{线路}）回单还差{原件/签章页}。
方便哪天交到{地点}？或先发清晰照片我预登记，原件{时限}前补上，避免压您这票结算。
```

## 3. 承运商 / 车队

```
{公司}您好，账期 {period} 对账：我方登记仍缺回单 {n} 票（明细附后）。
请协助催回或确认是否已寄出（快递单号）。回单齐且费用无争议的票我们优先安排结算核对。
明细：
1）运单… 线路… 金额… 状态…
```

## 4. 客户侧（回单在客户处）

```
{客户}您好，运单 {waybillNo}（{线路}）结账需回收单{原件/盖章件}。
是否仍在贵司？方便的话请{邮寄地址/扫描件邮箱}，我们收到后尽快完成对账。谢谢！
```

## 5. 仅电子、要原件

```
{对象}，{运单号} 电子回单已收到，按约定还需原件。请于 {日期} 前交回，电子件我先做预登记，原件到齐再销账。
```

## 6. 费用差异询证（非回单）

```
{对象}您好，运单 {waybillNo} 我方应付 ¥{a}，贵方账单 ¥{b}，差 ¥{diff}。
我方记录为{费用构成}。请帮确认是否含{等候/放空/…}或是否串票，方便我们本周内对平。谢谢！
```

## 7. 对内给老板的催办摘要

```
【回单催办摘要】
P0 {n} 票 · 预计卡结金额 ¥…
建议今日群催 + 私聊 Top {k} 司机。
话术已备，是否直接发群？
```
