# 回单对账作业 · 回单对账员

把回单回收情况和费用相关信息发给我，我帮你跟踪回单、汇总费用、生成对账单草稿并标注差异；需要催回单时生成提醒内容。

## 类型

Agent 型（单个 AI 专家）

## 功能

解决「回单回收慢、对账耗时长、易漏算错算」：

- **回单跟踪**：状态、形态、超期、持有人、P0/P1/P2 催收优先级  
- **费用汇总**：运费与附加费分项，按账期/对象切片  
- **对账单草稿**：匹配明细 + 无法匹配区 + 汇总  
- **差异标注**：标红阈值、受控原因码、建议付/暂缓/追差  
- **催办话术**：司机群、单人、承运商、客户  

附带 `pod-recon` Skill（字段、差异原因码、催办话术模板）。

与【物流作业领航】财务工位互补：本专家更聚焦 **回单闭环 + 对账单草稿 + 催办**；复杂 3PL 全能台仍可用领航。

## 使用示例

- 本周费用表 + 回单登记 → 对账单草稿、差异与催单话术  
- 承运商称缺 8 张回单 → 对齐真缺 vs 已回未登记  
- 月底总账差 1260 → 找差异票与可能原因  
- 群催超 7 天未交原件
