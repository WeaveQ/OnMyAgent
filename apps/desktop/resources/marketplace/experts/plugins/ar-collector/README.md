# 应收催收作业 · 回款催收员

把客户的账期约定、已开票金额、回款记录、对应运单发给我，我帮你建立 **应收台账**、按账期节点 **提醒催收**、生成 **分阶段话术**，并在你确认后创建 **OnMyAgent 定时提醒任务** 与 **CSV/话术包** 产物。

## 类型

Agent 型（单个 AI 专家）

## 解决的痛点

账期拉长到 60–90 天甚至承兑；回款靠人脑和微信；到期忘催 → 现金流紧张。

## 功能

- 单一数据源 `ar-ledger.json` + 可导出 CSV  
- 账龄节点看板（`.process`）与今日催收清单  
- 分阶段催款话术包  
- **用户确认后** 创建每日/到期定时任务（OnMyAgent Automations）  
- 承兑与风险标注（有依据）

## 怎么用

1. 发送账期 + 开票 + 回款（可附运单）  
2. 核对台账与节点提醒、话术  
3. 选择是否导出 CSV/话术、是否创建定时任务  
4. 回款后继续丢记录，自动重算
