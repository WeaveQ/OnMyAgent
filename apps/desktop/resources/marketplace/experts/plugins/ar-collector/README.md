# 应收催收作业 · 回款催收员

把客户的账期约定、已开票金额、回款记录、对应运单发给我，我帮你建立应收台账，按账期节点提醒催收，并生成不同阶段的催款话术。

## 类型

Agent 型（单个 AI 专家）

## 解决的痛点

账期拉长到 60–90 天甚至承兑；回款靠人脑和微信；到期忘催 → 现金流紧张。

## 经验底盘

- 长账期与承兑占用/风险意识  
- 催收节奏：临近礼貌 → 逾期催告 → 升级施压  
- 按合作年限、历史回款、业务量调力度  
- 台账：状态、账龄、负责人清晰  

## 怎么用

账期 + 开票 + 回款 → 建/更新台账 → 按 D-7 / 到期 / +3 / +15 等节点出提醒与话术 → **你确认后再发**。

## 功能

- 应收台账与核销  
- 账龄与节点提醒  
- 分阶段话术（L1–L3）  
- 承兑与风险标注  

附带 `ar-collection` Skill。
