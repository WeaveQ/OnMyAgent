# 分阶段催款话术与力度

## 力度档（对内先选档再套话术）

| 档 | 适用（有依据时） | 姿态 |
|----|------------------|------|
| L1 轻 | 老客户、历史稳、逾期短、仍在走货 | 提醒、给台阶 |
| L2 中 | 一般客户、首次逾期或偶发 | 正式、要明确付款日 |
| L3 重 | 长逾期、空头承诺多、余额大 | 升级措辞、对内建议停运/法务（须授权） |

**禁止**：人身威胁、深夜连续轰炸、泄露第三方隐私施压、未授权「已起诉」虚假表述。

## L1 · 账期临近（D-7）

```
{称呼}，您好。我司与贵司 {期间/票号} 运费账单余额约 ¥{余额}，按约定将于 {到期日} 到期。
方便预排付款吗？若已安排请告知大概到账日，我好做账。谢谢支持！
```

## L1/L2 · 到期当天

```
{称呼}，提醒一下：{票号/期间} 应付 ¥{余额} 今日到期。
请协助今日或明日安排电汇（如走承兑请告知票面到期日与金额）。回单/对账若有出入请直接说，我们马上核对。
```

## L2 · 逾期 +3

```
{称呼}，{票号} 已于 {到期日} 到期，目前仍有 ¥{余额} 未回。
请本周内给出明确付款日；若资金安排紧张，也可说明计划，我们好向财务同步。
望保持一直以来的合作节奏，谢谢。
```

## L2/L3 · 逾期 +15

```
{称呼}，{票号} 逾期已 {N} 天，余额 ¥{余额}。此前沟通记录：{摘要}。
请于 {明确日期} 前完成支付或给出盖章/书面付款计划。
逾期较久将影响后续账期与发运安排的内部评估（具体以我司通知为准）。请优先处理。
```

## 承兑场景

```
收到贵司承兑意向：请提供票面金额、到期日、是否可背书。
说明：承兑兑付前资金仍占用，若票期超过我司可接受范围，需协商贴现或缩短票期。兑付到账后我再销账。
```

## 老客户有分寸（逾期短）

```
{称呼}，合作一直很顺畅。这票 {票号} 刚过期几天，余额 ¥{余额}，麻烦抽空安排一下。
还在走货的部分我们正常保障；回款对齐后两边都轻松。您看这周哪天能定？
```

## 对内升级摘要（非直接发客户）

```
【催收升级建议】
客户… 余额… 逾期… 天
历史：… 力度建议：L3
可选拍板：① 限时书面计划 ② 新业务款到发货 ③ 法务函 ④ 暂停账期
话术已备，是否发送 L3 版？
```

## 回款确认

```
{称呼}，已核对到账 ¥{金额}（{日期}），对应 {票号} 已核销；剩余 ¥{余额}。
谢谢支持！
```
