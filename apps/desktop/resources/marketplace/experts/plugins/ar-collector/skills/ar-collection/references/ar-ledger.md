# 应收台账字段

## 账期规则 period.rule

| 示例 | 说明 |
|------|------|
| 月结，次月 N 日 | 固定付款日 |
| 对账确认后 net 30/60/90 | **起算日 = 双方确认对账日**（需记录） |
| 开票后 net 30 | 起算日 = 开票日 |
| 回单齐全后结算 | 卡回单状态（可与回单专家联动提示） |

约定不清 → 先追问，不擅自选起算日。

## 状态 status

| 值 | 中文 |
|----|------|
| open | 未到期 |
| due_soon | 临近到期（如 ≤7 天） |
| due_today | 到期当天 |
| overdue | 逾期 |
| partial | 部分回款仍有余额 |
| paid | 已结清 |
| disputed | 争议（对账未平） |
| written_off | 核销坏账（仅用户确认后） |

## 回款核销

1. 明确回款日期、金额、方式、对应票号（不明则「待认领」）。  
2. 优先匹配客户指定票；否则按约定（FIFO 或用户规则）。  
3. 更新 `amountPaid` / `amountOpen` / `status`。  
4. 承兑登记为「已收票未兑付」子状态，**兑付到账**后再计现金回清。
