# 应收催收技能（AR Collection）

把账期、发票、回款、运单收成 **可看、可催、可跟** 的应收台账与分阶段话术。

## 标准作业流程

1. **立规则**：账期怎么起算（对账确认日 / 开票日 / 月结固定日）、是否接受承兑。
2. **建台账行**：`references/ar-ledger.md` 开票、回款核销、余额、运单关联。
3. **算账龄与节点**：`references/aging-nodes.md` 默认 D-7 / 到期日 / +3 / +15。
4. **定力度**：`references/scripts-by-stage.md` 结合合作年限、历史回款、业务量。
5. **输出**：台账 + 今日催收清单 + 话术；承兑与风险单独标。
6. **回款后**：更新核销，重算余额与下一节点。

## 铁律

- 无来源不编造金额、票号、客户承诺。  
- 不威胁、不骚扰式催收；停运/法务升级须用户授权。  
- 承兑未兑付 ≠ 现金已回清。  
- 话术默认「你确认后再发」。
