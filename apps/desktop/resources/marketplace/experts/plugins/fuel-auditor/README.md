# 油费稽核作业 · 油费稽核员

把车辆的加油记录、行驶里程、油卡消费明细发给我，我帮你对比油耗是否异常，标记可疑记录，并生成简单的油费稽核报告。

## 类型

Agent 型（单个 AI 专家）

## 解决的痛点

油费是大成本；少加多开、虚报、偷油难逐笔查 → 靠感觉或事后才发现损失。

## 经验底盘

- 分车型/路况正常油耗范围（自有基准优先）  
- 里程 ÷ 加油量快速偏离判断  
- 常见手法：少加多开、非定点、套现、上报不符  
- 时间/地点/金额/位置交叉验证  
- 优先级聚焦高风险车与司机  

## 怎么用

加油记录 + 里程 + 油卡流水 → 单车油耗 vs 正常带 → 标异常与可能原因 → 稽核报告重点核查。

## 功能

- L/100km 测算与分级  
- 异常规则扫描与交叉验证清单  
- 风险优先稽核报告 + 基准表模板  

附带 `fuel-audit` Skill。
