# 油费稽核数据字段

## 加油 / 油卡流水

| 中文 | JSON | 说明 |
|------|------|------|
| 时间 | at | 加油时间 |
| 车牌 | plateNo | 主键 |
| 司机 | driver | 可选 |
| 油站 | station | 名称/品牌 |
| 是否定点 | isNetworkStation | 有名单才强判断 |
| 升数 | liters | 优先 |
| 金额 | amount | 无升数时用单价估算并标注 |
| 单价 | unitPrice | 可选 |
| 油品 | fuelType | 0# / 柴油等 |
| 支付 | payChannel | 油卡/现金/微信 |
| 流水号 | txnId | 去重 |

| 所属 | fleet | 自有/外请 |

## 对齐规则

1. 同一 `plateNo` 按时间排序加油事件。  
2. 段里程 = 相邻两次有效码表差；缺码表则用用户给的区间里程。  
3. 段加油量 = 该段内加油升数合计（定义写进报告：按「加满到加满」或「自然周」）。  
4. 冲突票（同一 txn 重复）去重；对不上车牌进 **待确认**。

## 油耗公式

- `L/100km = (liters / distanceKm) * 100`  
- `distanceKm ≤ 0` → 不计算，标数据异常  
- 仅有金额无升数：`liters ≈ amount / unitPrice`（必须标注估算）
