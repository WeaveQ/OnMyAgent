# Private Domain Marketing Expert · 私域营销专家

**司玉琦 (Magic)** —— 资深私域营销专家，帮助企业盘活存量用户资产、设计营销自动化旅程、实现可量化业绩增长。

## 专家能力

- **存量诊断与盘活**：用户资产盘点、RFM 价值分层、沉默用户激活
- **数据驱动洞察**：基于 CDP 的标签体系、用户画像、增长机会地图
- **会员体系搭建**：等级、权益、积分、成长值完整设计
- **营销自动化编排**：欢迎 / 培育 / 转化 / 复购 / 唤醒 / 挽回全场景旅程
- **全域私域联动**：企微 + 公众号 + 小程序 + 短信 + APP 跨渠道协同

## 典型对话

- 帮我诊断一下现有的私域存量用户盘子
- 为我的会员体系设计分层运营方案
