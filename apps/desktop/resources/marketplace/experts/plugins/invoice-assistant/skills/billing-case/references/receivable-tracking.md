# 回款跟踪与催办节奏

## 原则

1. **先对账后催款**：开票与运单对应不齐时，先核对再催，避免客户用金额异议反推。
2. **节奏分层**：账期内软催、到期前硬催、逾期即升级；不同阶段话术不同。
3. **留痕**：催办走书面（微信/邮件），口头后补文字确认。
4. **不伤关系**：软催给理由、硬催给期限、升级给后果，措辞有分寸。

## 账期节点表

| 节点 | 时机 | 方式 | 目的 |
|------|------|------|------|
| 开票完成通知 | 开票当日 | 软 | 告知已开、附回执，建立回款倒计时 |
| 账期过半提醒 | 账期 1/2 时 | 软 | 确认客户收到票、内部流程已走 |
| 到期前提醒 | 到期前 3-5 天 | 中 | 提醒付款安排，预留审批时间 |
| 到期当日 | 到期日 | 中 | 礼貌确认付款进度 |
| 逾期 3 天 | 逾期 3 天 | 硬 | 明确逾期、给付款期限 |
| 逾期 7 天 | 逾期 7 天 | 硬 | 升级对接人、留痕 |
| 逾期 15 天+ | 逾期 15 天+ | 升级 | 升级商务/法务、控货/控额度 |

## 催办输出格式

```markdown
### 回款催办卡
- 客户：…
- 批次/票号：…
- 开票日 / 账期 / 到期日：…
- 应收金额：…
- 当前节点：…
- 建议方式：软催 / 硬催 / 升级
- 话术要点：…
- 下次跟进日：…
```

## 加速动作（示例）

- 开票完成即通知客户并提供回执，缩短内部流转认知差
- 账期过半主动确认「票已收到、流程走到哪」，避免到期才发现卡内部审批
- 逾期话术同时发给对接人 + 抄送其上级，增加推进动力
- 长期合作客户控额度：未回款超阈值暂停新发运，用业务节奏倒逼回款
- 月结客户固定每月 X 号对账、Y 号开票、Z 号催款，形成节奏

## 升级判断

- 逾期超 30 天且沟通无效 -> 建议商务介入谈分期或抵扣
- 逾期超 60 天或金额过大 -> 建议法务发函
- 客户失联/经营异常 -> 建议立即停运并保全证据
- 本专家只给节奏建议，最终催收动作由你拍板
