# 开票申请模板与催款话术

## 1. 开票申请内容（提交财务）

```
【开票申请】
客户名称（抬头）：…
税号：…
地址：…
电话：…
开户行：…
银行账号：…
票种：增值税专用发票 / 普通发票 / 电子普票
货物或服务名称：…
金额（含税）：…  税率：…%
对应运单号：
  1. YDxxxx  …元
  2. YDxxxx  …元
  …
合计：… 元（共 N 票）
账期：月结 … 天
备注：…
申请人 / 日期：…
```

## 2. 开票完成通知客户（软）

```
{称呼}，您好。贵司 {批次/月份} 运费已开票完成：
发票类型：…  金额：…  票号：…
账期到期日：…
电子票已发至 {邮箱} / 纸质票已寄出（单号：…）。
如有疑问请反馈，无异议请按账期安排付款，感谢。
```

## 3. 账期过半确认（软）

```
{称呼}，打扰了。{批次} 发票已于 {日期} 开出，想确认下：
1）票是否收到
2）内部流程走到哪一步
便于我们对齐回款节奏，感谢配合。
```

## 4. 到期前提醒（中）

```
{称呼}，您好。{批次} 发票账期将于 {日期} 到期，应付款 … 元。
为避免逾期影响后续合作，烦请安排付款。如有审批进度，劳烦同步一下，谢谢。
```

## 5. 逾期催款（硬）

```
{称呼}，{批次} 发票已逾期 {N} 天，应付款 … 元（票号 …）。
按合同约定已构成逾期，请于 {期限，如 3 个工作日} 内安排付款。
如遇特殊情况请书面说明，否则我司需按流程处理。
```

## 6. 逾期升级（对接人 + 上级）

```
{对接人} {上级}：
{批次} 发票逾期 {N} 天，金额 … 元，前期已多次沟通暂未回款。
为维护合作，希望贵司协助推进，于 {期限} 前明确付款计划。
如仍无法推进，我司将按合同启动相应措施，望理解。
```

## 7. 金额异议回复

```
{称呼}，收到金额异议反馈。
我司核对运单-开票对应表如下：
  运单号 | 金额 | 说明
  …
合计 … 元，与开票金额一致。
请指出差异票号，我司立即复核。核对完成前暂不开票/暂不调整，感谢配合。
```

## 8. 抬头变更通知确认

```
{称呼}，收到贵司开票主体变更为 {新公司全称}。
请提供新主体开票资料：名称、税号、地址电话、开户行账号（扫描件为准）。
1）旧主体未开完的运单：单独开旧抬头
2）变更日后的运单：开新抬头
新旧不可混开。新资料收到后我司更新档案。
```

## 禁止

- 威胁、侮辱、教唆不当施压
- 未核实就承认金额错误、随意核减
- 替你决定最终抬头与税率选择
- 编造金额、单号、日期
