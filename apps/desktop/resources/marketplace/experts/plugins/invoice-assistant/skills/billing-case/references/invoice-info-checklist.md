# 开票信息完整清单

每项标记：`已有` / `缺失` / `待确认` / `不适用`。

## 通用（所有票种）

| 信息 | 说明 |
|------|------|
| 客户名称（抬头） | 营业执照全称，不可简写 |
| 税号 | 统一社会信用代码 / 纳税人识别号 |
| 地址 | 开票地址 |
| 电话 | 开票联系电话 |
| 开户行 | 银行全称（到支行） |
| 银行账号 | 对公账号 |
| 货物或服务名称 | 统一归类后的品名 |
| 金额 | 含税/不含税需注明 |
| 税率 | 13% / 9% / 6% / 3% / 1% 等 |
| 对应运单号 | 本批运单号清单 |

## 增值税专用发票（额外）

| 信息 | 说明 |
|------|------|
| 是否一般纳税人 | 客户资质 |
| 开票资料三证 | 营业执照、开户许可证、税务登记（视需） |
| 注册地址与电话 | 必须与税号登记一致 |
| 开户行与账号 | 必须准确，影响抵扣 |

## 普通发票 / 电子普票

| 信息 | 说明 |
|------|------|
| 抬头 | 个人或单位全称 |
| 税号 | 单位需提供；个人普票可不填 |
| 接收方式 | 邮寄地址 / 邮箱（电子票） |

## 运单-开票对应

| 信息 | 说明 |
|------|------|
| 运单号清单 | 本批全部运单号 |
| 单票金额 | 每票对应金额（便于核对） |
| 合计金额 | 各票求和，与开票金额一致 |
| 线路 | 便于按线路汇总 |
| 货品归类 | 运输服务 / 货物 / 技术服务费等 |

## 资料收集要点（给业务）

1. 新客户首次开票前，一次性收齐「名称 + 税号 + 地址电话 + 开户行账号」存档。
2. 抬头信息以客户提供的开票资料扫描件为准，不凭记忆。
3. 客户开票主体变更时，单独走「抬头变更」流程，新旧不可混开。
4. 运单金额异议先核对运单-开票对应表，定位差异票号再处理。
5. 货物或服务名称归类与客户合同一致；归类不确定时问商务。

## 资料包目录模板

```text
BILL-{customer}-{batch}/
  01-客户开票资料/
  02-运单清单与金额/
  03-开票申请/
  04-开票完成回执/
  05-寄送与签收/
  06-回款记录/
  07-沟通与异议/
```
