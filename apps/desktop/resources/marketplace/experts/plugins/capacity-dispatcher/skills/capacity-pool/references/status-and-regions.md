# 状态、方向与车型口语对照

## 司机 / 车辆状态（status）

| 规范枚举 | 中文展示 | 常见口语 | 是否计入“当前可用” |
|---------|----------|----------|-------------------|
| empty_idle | 空车待派 | 空了、在家等活、停车待机、有空 | 是（需 fresh） |
| partial_load | 有余量可配 | 还能配一点、半车、顺路配货、剩几方 | 是（写明剩余吨方） |
| en_route_unload | 在途待卸 | 还在路上、快到了、卸完再说、到仓排队 | 条件可用（看 freeAt） |
| loading | 装货中 | 在装、等装、厂里装货 | 一般否，除非声明装完时间 |
| busy_full | 已满/已接单 | 满了、接了下一票、今天不拉了 | 否 |
| off_duty | 休息/下班 | 休息、回家了、明天再出 | 否 |
| maintenance | 维保/事故 | 修车、年审、故障 | 否 |
| unknown | 未知 | 含糊、无法判断 | 否，需追问 |

## 预计空闲时间（freeAt）口语

| 口语 | 规范处理 |
|------|----------|
| 已经空了 / 现在有空 | freeAt = 已空闲；status 倾向 empty_idle |
| 再过一小时卸完 | 换算为具体时钟（若知当前时间），否则记“约 1 小时后卸完” |
| 今晚 10 点卸完 | freeAt = 当晚 22:00（写明日期） |
| 明天上午才能走 | freeAt = 次日上午；status 可 off_duty 或 en_route_unload |
| 不确定 / 看情况 | freeAt 未知，追问时间窗 |

## 剩余载重 / 方数口语

| 口语 | 处理 |
|------|------|
| 空车 / 全空 | residual = 额定；若无额定则 residual 记“满额未知”，追问车型吨方 |
| 还能拉 8 吨 | residual.loadT = 8 |
| 剩 3 方 | residual.volumeCbm = 3 |
| 还能拉一点 / 半车 | 置信度中/低，必须追问大约吨或方 |
| 顺路小货 | 倾向 partial_load，追问上限与方向 |

## 方向偏好（preferredLanes）

| 口语 | 规范 |
|------|------|
| 回深圳 / 想回广州 | preferredLanes = ["深圳"] / ["广州"] |
| 东莞本地短途 | preferredLanes = ["东莞本地"] |
| 去哪个都行 / 随便 | preferredLanes = ["any"] |
| 不接返程以外的 | 写明仅返程方向，其它方向匹配时降权 |
| 不跑高速 / 不进市区 | 记入 remarks 或 capabilities 限制 |

## 车型常见说法

| 口语 | 规范示例 |
|------|----------|
| 4 米 2 / 金杯 / 依维柯 | 4.2 米厢货（或按用户原文保留品牌） |
| 6 米 8 高栏 | 6.8 米高栏 |
| 9 米 6 / 13 米 | 9.6 米 / 13 米（补厢型：厢/高栏/平板） |
| 尾板车 | 在 capabilities 加“尾板”，车型仍写车长厢型 |
| 冻柜 / 冷藏 | capabilities 加“冷藏”，追问温区 |

## 区域归并（展示用）

- 列表查询默认按 **城市 → 区/镇** 分组。
- 同城多点（茶山、常平、寮步）可归入“东莞东部”等自定义片区，但表格仍保留原文位置。
- 仅有“广东”“华南”时标记位置过粗，催更到市/区。

## 习惯与表现（仅有依据时）

- 可记录：常跑线路、常停靠园区、是否爱接配货、历史准点/拒单备注（用户或系统提供时）。
- **禁止**无数据时打“靠谱分”或编造星级。
