# 运力池话术模板

## 原则

1. **一次问清**：位置、剩余吨/方、状态/空闲时间、可接方向、车型关键能力一次要齐。
2. **给填空/选择题**：司机回复成本低。
3. **可直接转发**：称呼自然，适合微信群或私聊。
4. **催更与点名分开**：群广播求更新 vs 点名具体司机。

## 1. 司机群点名 / 回报话术

```
各位师傅好，麻烦快速回一下当前运力（复制填空即可）：
1）现在位置：
2）车型/车牌：
3）空车还是有货？剩余大约几吨/几方：
4）预计什么时候空闲：
5）能不能接往{方向}的货（能/不能/顺路可以）：
谢谢，回了的我马上记进今天运力表。
```

## 2. 定向方向招车

```
有一票{货物简述}，{装货地}→{卸货地}，约{吨/方}，要{车型/尾板等}，{时效}。
现在在{区域}附近、{时间窗}能到的师傅回一下「位置+剩余吨方+车牌」，我按顺序联系。
```

## 3. 超时未更新催更（群）

```
运力表里这些师傅超过{N}小时没更新了，还在跑的回个位置和是否有空，不方便的回「休息」我先标掉，避免瞎派：
{名单：师傅/车牌/上次位置/上次更新时间}
```

## 4. 单人催更（私聊）

```
{师傅}，您上次报到在{位置}（{更新时间}），表上有点旧了。
现在还在那边吗？空车吗？大概还能拉几吨/几方？预计啥时候有空？回四个数就行。
```

## 5. 信息不全追问（单人）

```
{师傅}收到，还差两点好记表：
1）剩余大概几吨或几方？
2）预计什么时候空（或已空闲）？
{可选：能不能接{方向}？}
```

## 6. 匹配结果转达调度（内部）

```
候选运力（按匹配度）：
1）{车牌/师傅} | {位置} | {车型} | 余{吨/方} | {状态}/{空闲} | 方向{...} | 新鲜度{fresh/aging} | 说明{命中/缺口}
2）...
已过期未列入：{名单}。建议先确认第 1 名是否接单再锁车。
```

## 7. 向司机确认是否接票（私聊）

```
{师傅}，有一票{装}→{卸}，约{吨/方}，{特殊要求}，计划{发货时间}。
您这边{位置/余量}对得上吗？能接回「能+最早几点到装货地」，不能回「不能」就行。
```

## 8. 看板开头摘要模板

```
【运力看板 {日期 时间}】
可用（fresh）：{n} 台 | 将过期（aging）：{n} 台 | 已过期待催（stale）：{n} 台
按区域：{区域A n 台}；{区域B n 台}
下文表格为明细；标红/stale 不参与派车推荐。
```
