# 税务申报准备 & 会计师交接

## 概述

围绕中国小企业的常规税务节奏（月度/季度增值税申报、季度企业所得税预缴、年度汇算清缴、个税代扣、社保公积金）做材料归集、抵扣项梳理、自查和会计师交接打包。

**重要提醒：本技能负责整理和准备，不替代税务师/注册会计师出具正式意见。所有输出末尾必须附上："具体税务处理请与代账会计师或税务师确认后再申报。"**

## 工作流

### Step 1: 纳税人身份与基础信息核对

先确认企业当前的税务身份（影响后续所有处理逻辑）：

- [ ] 增值税身份：小规模纳税人 / 一般纳税人
- [ ] 企业所得税征收方式：查账征收 / 核定征收
- [ ] 适用的税收优惠：小型微利企业、研发费用加计扣除、行业专项优惠等
- [ ] 主管税务局所在地、电子税务局账号是否可登录
- [ ] 法人/财务负责人/办税员实名认证状态

身份不同，下面所有节奏和阈值都会变（如小规模季度免征额、小微企业所得税优惠率），不要套用模板。

### Step 2: 发票合规自查

发票是中国税务的核心，先把发票这关过了：

**进项发票（一般纳税人才涉及抵扣）：**
- [ ] 已认证抵扣的进项专票清单（金税系统/电子税务局导出）
- [ ] 未认证的进项专票是否在 360 天有效期内
- [ ] 异常凭证、走逃失联企业开具的发票筛查
- [ ] 普票与专票分类，普票不得用于抵扣
- [ ] 不得抵扣项目核查（集体福利、个人消费、免税项目对应进项）

**销项发票：**
- [ ] 已开发票汇总（含红冲），与营收对账
- [ ] 待开票订单清单（合同已履行但未开票）
- [ ] 开票内容是否与实际业务一致（避免虚开风险）
- [ ] 开票限额、领购数量是否够用

**发票存档：**
- [ ] 纸质发票按月装订
- [ ] 电子发票 PDF + OFD 双份归档
- [ ] 全电发票（数电票）查验记录

### Step 3: 申报材料归集

**增值税申报材料（小规模季度 / 一般纳税人月度）：**
- [ ] 销售收入明细（区分征税、免税、出口、视同销售）
- [ ] 进项税额抵扣明细（一般纳税人）
- [ ] 当期销售额是否触发小规模免征额阈值
- [ ] 附加税费（城建税、教育费附加、地方教育附加）

**企业所得税材料（季度预缴 + 年度汇算清缴）：**
- [ ] 利润表（季度预缴用）
- [ ] 收入、成本、费用、税金明细
- [ ] 资产折旧/摊销明细
- [ ] 弥补以前年度亏损情况
- [ ] 小型微利企业判定（资产、人数、应纳税所得额三个条件）
- [ ] 研发费用归集表（如适用加计扣除）

**个人所得税（工资薪金代扣代缴）：**
- [ ] 员工工资表、奖金、补贴
- [ ] 专项附加扣除信息（自然人电子税务局 WEB 端导出）
- [ ] 全年一次性奖金计税方式选择记录
- [ ] 劳务报酬、稿酬等其他所得明细

**社保公积金：**
- [ ] 社保缴费基数核定单（年度调整）
- [ ] 五险一金月度缴费明细
- [ ] 公积金增减员记录
- [ ] 工伤事故/生育报销等异动事项

### Step 4: 可抵扣 / 可税前扣除项梳理

按企业所得税口径，审计常被漏掉的扣除项：

| 类别 | 检查要点 | 易漏 |
|------|---------|------|
| 业务招待费 | 不超过当年营收 5‰，且实际发生额 60% | 与会议费混淆 |
| 广告与业务宣传费 | 一般行业 15% 限额，化妆品/医药/饮料 30% | 未按行业取限额 |
| 职工福利费 | 不超工资总额 14% | 节日礼品、过节费 |
| 工会经费 | 不超工资总额 2% | 未实际拨缴 |
| 职工教育经费 | 不超工资总额 8%（高新技术企业可全额） | 内训、外训、考证 |
| 公益性捐赠 | 不超利润总额 12%，超过部分结转 3 年 | 票据合规性 |
| 研发费用加计扣除 | 现行政策 100% 加计 | 人员人工、直接投入归集口径 |
| 固定资产折旧 | 加速折旧政策 | 500 万元以下设备一次性扣除 |
| 资产损失 | 报送清单备查 | 坏账核销、存货报废 |
| 借款利息 | 关联方债资比限制（金融企业 5:1，其他 2:1） | 非金融机构借款利率上限 |

每一项标明扣除金额、依据文件、备查资料路径。

### Step 5: 自查质检

交给代账会计/税务师之前先自己过一遍：

- [ ] 增值税申报销售额与利润表收入差异（关注未开票收入、视同销售）
- [ ] 企业所得税"利润总额"与利润表数据一致
- [ ] 社保人数与个税申报人数大致匹配（差异过大易被税务局预警）
- [ ] 工资表合计数与个税申报系统中合计一致
- [ ] 增值税与附加税费计算关系正确
- [ ] 季度预缴累计金额勾稽
- [ ] 没有把私人消费、股东个人支出混入企业费用
- [ ] 关联交易已按规定填报

### Step 6: 会计师 / 代账机构交接包

完整交接清单：

1. **业务说明函** —— 本期重大事项（注册地变更、股权变动、新业务线、大额资产购置、重大合同）
2. **发票包** —— 进项发票清单 + 销项发票清单（含红冲）
3. **凭证与账簿** —— 总账、明细账、现金/银行日记账
4. **银行流水** —— 对公账户全年流水（按月）
5. **工资与个税** —— 工资表、个税申报表、社保公积金缴费明细
6. **合同清单** —— 大额合同 + 关联交易合同
7. **资产与折旧** —— 新增固定资产清单、折旧表
8. **税前扣除明细** —— 各扣除类别逐项工作底稿
9. **历史申报** —— 上年度汇算清缴报告 + 本年已申报的季度表
10. **待确认问题** —— 税务师/会计师需要拍板的事项

电子化交付，按上述目录建文件夹，外发前打包加密。

### Step 7: 申报后归档

申报完成后：

- [ ] 申报表、完税凭证 PDF 归档（**至少保存 10 年**，账簿凭证按《会计档案管理办法》）
- [ ] 完税金额与到账银行流水对账
- [ ] 在现金流预测中登记下一期预缴税款
- [ ] 在日历中设好下次申报截止日提醒
- [ ] 根据税务师反馈，修正日常记账口径
- [ ] 启动下一周期的发票、凭证归档文件夹

## 中国主要申报节奏

| 税种 | 节奏 | 申报截止日 |
|------|------|----------|
| 增值税（小规模） | 季度 | 季度终了次月 15 日 |
| 增值税（一般纳税人） | 月度 | 次月 15 日 |
| 企业所得税预缴 | 季度 | 季度终了次月 15 日 |
| 企业所得税汇算清缴 | 年度 | 次年 5 月 31 日前 |
| 个人所得税（工资薪金）| 月度 | 次月 15 日 |
| 个税年度汇算 | 年度 | 次年 3 月 1 日 — 6 月 30 日 |
| 社保公积金 | 月度 | 各地不同，一般 15-25 日 |

遇周末或法定节假日顺延，以电子税务局公告为准。

## Notes

- 本技能不替代税务师/代账会计——做好整理工作，让专业人员高效推进
- 所有税务输出必须附免责说明："具体税务处理请与代账会计师或税务师确认后再申报。"
- 多地经营/跨省经营涉及总分机构汇总申报，按 [国税发〔2012〕57 号] 等政策处理，需单独评估
- 政策变化频繁（如小规模免征额、小微企业认定标准），最终以税务总局当年公告为准
