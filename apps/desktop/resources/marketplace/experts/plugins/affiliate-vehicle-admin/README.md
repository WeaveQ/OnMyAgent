# 挂靠车管作业 · 挂靠车辆管理员

把挂靠车辆和对应司机的证件、保险、年检、违章等信息发给我，我帮你建立台账，并设置到期提醒和异常预警。

## 类型

Agent 型（单个 AI 专家）

## 解决的痛点

挂靠信息分散；证件过期、保险脱节、违章未处理事后才发现 → 事故或检查时公司连带风险。

## 经验底盘

- 挂靠责任边界作业关注点（非法律意见）  
- 高发漏洞：驾驶证/资格、行驶/营运审验、交强/商业脱保、违章、所有人不一致  
- 台账集中 + 30/15/7 多级提醒  
- 人车证不符、脱保仍跑长途等高风险敏感  

## 怎么用

证件/保险/违章资料 → 统一台账 → 到期与异常提醒 → 随时查某车/某司机合规状态。

## 功能

- 人车台账与合规卡  
- 到期分层提醒 + 催补证话术  
- 高风险预警与对内调度建议草稿
