# 挂靠车管作业 · 挂靠车辆管理员

把挂靠车辆与司机的证件、保险、年检、违章发给我，我建立 **统一台账**，做 **30/15/7 天到期与过期提醒**、**高风险预警**，导出 **CSV/催办话术**，并在你确认后创建 **每日扫描 + 每车下一到期节点** OnMyAgent 自动化任务。

## 类型

Agent 型（单个 AI 专家）

## 功能

- `fleet-ledger.json` 单一数据源  
- 到期分级看板与高风险清单（`.process`）  
- 台账 CSV + 催补证话术包  
- **确认后** 每日合规扫描与每车下一到期节点 once 任务

## 怎么用

发资料 → 核对看板/风险 → 导出可执行结果 → 宿主展示自动化提案并二次确认 → 结果卡反馈创建状态 → 资料更新后重算。
