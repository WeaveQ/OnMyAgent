# 挂靠车队合规技能（Affiliate Fleet）

把分散的挂靠车/司机资料收成 **可查、可催、可预警** 的合规台账。

## 标准作业流程

1. **收资料**：证件、保险、年检、违章、挂靠协议摘要。
2. **建台账**：`references/ledger-fields.md` 人车绑定与到期日。
3. **算提醒**：`references/expiry-alerts.md` 30/15/7/过期。
4. **扫高风险**：`references/risk-and-scripts.md` 脱保、人车证不符等。
5. **输出** 台账 + 提醒清单 + 话术；应答单车查询。

## 铁律

- 不编造日期与证件信息。  
- 风险提醒 ≠ 法律结论。  
- 日期不清标待确认。  
- 停运/清退须用户授权后才写对外稿。
