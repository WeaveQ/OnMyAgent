# 高风险项与催办话术

## 风险码

| 码 | 情形 | 说明 |
|----|------|------|
| DOC_EXPIRED | 证照已过期 | 驾驶证/资格/年检/营运审验 |
| INS_LAPSED | 交强或商业脱保/过期 | 交强优先标严重 |
| INS_SOON_LONGHAUL | 保险将到期仍长途 | D-15 内 + 在途/计划长途 |
| PERSON_VEHICLE_MISMATCH | 人车证不一致 | 司机准驾/绑定与车不符 |
| OWNER_MISMATCH | 登记所有人≠实际/挂靠登记 | 信息冲突 |
| VIOLATION_PILE | 违章长期未处理 | 条数或时长超你的阈值（默认未处理≥3 或 >30 天） |
| DATA_GAP | 关键字段缺失 | 无法判断可否派车 |

## 高风险输出表

|--------|-----------|--------|------|------------------|

## 话术 1 · 司机群催补证（D-30/D-15）

```
各位师傅，挂靠台账显示以下证件/保险将在 30 天内到期，请尽快办理并回传新有效期照片，避免影响派车：
1）{车牌} {司机} {项目} 到期 {日期}
…
回「已办+新到期日」我好销提醒。谢谢！
```

## 话术 2 · 单人私聊（D-7/过期）

```
{师傅}，您绑定的 {车牌} {项目} 已{到期/仅剩N天}。
请于 {日期} 前完成续期并把新证件/保单止期发我；逾期未更新将按公司规定限制派车（最终以调度通知为准）。
```

## 话术 3 · 违章催办

```
{师傅}，{车牌} 名下未处理违章 {n} 条（如有明细附后）。
请尽快处理并回执，避免审验/年检和路检麻烦。处理完发截图我更新台账。
```

## 话术 4 · 对内调度（保险将到期仍在途）

```
调度旁：{车牌} 商业险/交强止期 {日期}（剩余 {n} 天），当前任务 {线路/位置}。
建议评估：就地控风险 / 换车接驳 / 加速返程后续保。是否按高风险限制新派长途，请老板拍板。
```

## 禁止

- 伪造年检、假保单建议  
- 未授权对外宣称「交警已扣车/已起诉」  
- 泄露无关司机隐私用于威胁
