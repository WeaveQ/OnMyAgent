# 到期提醒分级

## 默认阈值（可改）

| 级别 | 条件 | 展示 |
|------|------|------|
| expired | 剩余天 < 0 | 已过期 · 红 |
| D-7 | 0 ≤ 剩余 ≤ 7 | 7 天内 · 红/橙 |
| D-15 | 8 ≤ 剩余 ≤ 15 | 15 天内 · 橙 |
| D-30 | 16 ≤ 剩余 ≤ 30 | 30 天内 · 黄 |
| ok | > 30 | 正常 |

对 **每一证件/保险止期字段分别** 计算，取最严重级别作为该车「证件险」综合色，并列出明细。

## 提醒清单输出

```markdown
### 到期提醒 {asOfDate}
#### 已过期
| 车牌 | 司机 | 项目 | 到期日 | 过期天 |
#### D-7
…
#### D-15
…
#### D-30
…
```

## 多级触达建议（对内）

| 级别 | 建议动作 |
|------|----------|
| D-30 | 首次礼貌催补证/续保 |
| D-15 | 二次催办 + 登记承诺日期 |
| D-7 | 升级调度：限制新派长途（须你确认） |
| expired | 高风险：建议停派直至更新（须你确认） |

## 保险止期 + 在途

若商业险/交强险 ∈ D-15 或更紧，且用户说明「仍在长途/返程」→ 强制进入高风险清单，提示换车或就地停等方案（作业建议）。
