# 物流作业领航

> **像微信一样给财务、调度、业务、异常派活。**  
> 丢账单、丢截图，直接出差异表、报价单和话术。

## 是什么

**物流作业领航** 是 OnMyAgent 专家包，服务 **中小 3PL / 专线 / 车队** 老板与调度台。

一个专家入口，四个数字工位：

| 工位 | 本事 |
|------|------|
| 财务小账 | 承运商账单 vs 应付、回单票差、催收/付款清单 |
| 调度阿强 | 派车复述、调度早报、风险单 |
| 业务报价 | 专线/整车报价结构、底价与成交话术 |
| 异常台 | 破损延误拒收、责任初判、索赔闭环与话术 |

## 适合谁

- 专线 / 落地配 / 区域 3PL 老板  
- 调度、信息部、货运部对账与异常处理  
- 需要「对话派活」而不是复杂 TMS 菜单的人  

**不适合：** 工厂直客供应链全局优化、采购寻源（请用供应链/采购类专家）、法律终审意见。

## 内置 Skills

| Skill | 说明 |
|-------|------|
| `recon-ap-ar` | 对账台 |
| `exception-claim-handler` | 异常索赔 |
| `freight-quote-desk` | 报价台 |

## 怎么用

直接说：

- 「这是承运商账单，跟应付表对一下，差超 50 标红」  
- 「上海→成都 12 方 1.8 吨 48h 专线，给报价结构」  
- 「货主说破损司机说没有，帮我初判和回复」  
- 「写今天调度早报」  

## 边界

- 无底价库时不编造「全网真实成交价」  
- 不协助超限、假回单、逃税等违法操作  
- 改账 / 对外认责 / 催款发送 → 必须老板拍板  
- 索赔金额与责任为作业建议，非法律意见
