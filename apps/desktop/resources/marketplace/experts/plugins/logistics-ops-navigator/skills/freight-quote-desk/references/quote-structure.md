# 报价结构标准项

按需出现，不适用填 `—` 或「含在干线」。

| 费用项 | 说明 |
|--------|------|
| 干线运费 | 专线/整车主体 |
| 提货费 | 门到港/网点 |
| 送货费 | 港到门 |
| 上楼/进仓 | 有则单列 |
| 多点提送 | 点数与加价 |
| 回单费 | 纸质/电子 |
| 保险/保价 | 费率或定额 |
| 等通知/压车 | 超时规则 |
| 放空风险金 | 若约定 |
| 信息费/渠道费 | 若有，对内口径可与对客分离 |
| 其他 | 写明 |

## 计费口径提示

- **重货**：常按吨；**泡货**：常按方；给 **折算吨** 时写清公式  
- 整车：按车型+公里/包车，不与零担单价混用  
- 总价 = 各项之和；折扣写在「成交带」不要偷偷改表  

## 示意数字

凡非用户提供的成本/历史价，单元格或脚注必须写 **`示意`**。
