# 话术骨架（可改数字与称呼）

## 给货主（稳情绪 + 要证据）

```text
{称呼}您好，这票{运单/线路}的情况我们已登记。
目前已掌握：{已知}。
为尽快处理，还请补充：{缺口1}、{缺口2}。
我们会在收到材料后 {时限，如24小时内} 给您书面处理意见，请放心。
```

## 给司机/承运（要事实，不先骂）

```text
师傅/调度，{运单}货主反馈{争议点}。
请您今天内提供：{装车照/现场照/说明}。
有约定的话把聊天或合同条款截图一并发我，方便对责，避免双方吃亏。
```

## 内部（给老板）

```text
风险：{高/中/低}。主因倾向：{...}。
建议：{补证 / 协商区间 / 暂扣运费}。
不宜：{例如立刻全额认赔或公开撕逼}。
```

语气规则：不威胁、不羞辱、不伪造「已报警」等事实。
