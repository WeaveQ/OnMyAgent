# 差异表字段说明

## 主表列

| 列 | 必填 | 说明 |
|----|------|------|
| # | 是 | 序号 |
| 运单号 | 尽量 | 对齐键 |
| 线路 | 建议 | 如 上海→成都 |
| 我方应付 | 是 | 我方系统/表 |
| 对方金额 | 是 | 账单 |
| 差异 | 是 | 对方 − 我方（表头写清） |
| 回单 | 建议 | 已回/未回/不合格 |
| 原因初判 | 是 | 使用 reason-codes |
| 建议 | 是 | 付 / 暂缓 / 追差 / 人审 |

## 标红规则

默认 `|差异| > 阈值` 在「差异」列用 **粗体** 或标注 `⚠`。

## 排序

1. 标红差异从大到小  
2. 回单未回  
3. 其余
