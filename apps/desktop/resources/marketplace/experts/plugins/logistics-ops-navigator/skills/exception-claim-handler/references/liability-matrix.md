# 责任初判参考（作业级，非法学判决）

## 常见场景 → 优先核对

| 场景 | 先查 | 承运侧常见点 | 托运侧常见点 |
|------|------|--------------|--------------|
| 包装破损 | 装车照片、包装约定 | 绑扎/雨布/野蛮卸 | 包装不足、隐患货 |
| 短少 | 装车件数、铅封、中转 | 中转失控 | 报错件数 |
| 延误 | 约定时效、发车晚、路况 | 擅自绕行/晚点发 | 货没备好、等单 |
| 压车 | 到达时间、卸货窗口 | 未预约/迟到 | 无仓位、晚班卸 |
| 放空 | 取消时间、是否已发车 | 已出车取消规则 | 货主取消过晚 |
| 拒收 | 拒收原因、是否变质/错货 | 送错/延误致损 | 无理由拒收、单证 |

## 输出要求

- 至少给 **主责倾向 + 次要可能**  
- 每条绑定 **证据或缺口**  
- 出现人身伤害、危险品、重大火灾等 → 建议停线上争辩，走保险/报警/法务，不在此包内「谈妥」
