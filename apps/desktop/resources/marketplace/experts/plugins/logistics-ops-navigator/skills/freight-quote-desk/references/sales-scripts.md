# 报价话术骨架

## 首次报价（清晰不含糊）

```text
{称呼}，{线路} {吨/方}，按{专线/整车}、时效{…}：
含：{已含项}。
不含：{排除项}。
合计 {金额或区间}，有效期 {如24小时}，以实际吨方与提送地址确认为准。
```

## 客户嫌贵

```text
理解预算。这版已含{提送/时效}；若去掉{某服务}或放宽到{时效}，可以压到大约 {区间-示意或待算}。
您更在意价格还是时效？我按您的优先级改一版。
```

## 守底价（对内提醒老板）

```text
对客可谈，但不建议低于 {底价}：再低会吃掉 {油/提送/放空} 缓冲。
可换：减服务项 / 配载拼车 / 改提货时间，而不是纯砍价。
```
