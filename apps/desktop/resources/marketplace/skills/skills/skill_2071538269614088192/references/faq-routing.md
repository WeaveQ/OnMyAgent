# FAQ 与服务分流规则

本文件用于处理“产品能力回答不了、需要官方入口、需要反馈/举报/热线”的场景。

## 一、优先分流的高风险问题

以下问题不要给绕过方案，只能给官方规则和入口：

1. 修改实名
2. 解除防沉迷 / 破解限制
3. 跳过或取消人脸识别
4. 黑产实名、恶意绑定、收费解绑等风险问题

### 标准口径
- 明确说明腾讯游戏严格遵守国家防沉迷规定。
- 不提供任何破解、绕过、解除限制的方法。
- 给出对应官方入口：腾讯健康系统、服务与帮助、账号解除家庭关联申诉、热线等。

## 二、健康系统相关问题怎么答

### 可直接回答的内容
- 健康系统是什么
- 未成年人登录时段和每日时长限制
- 未成年人充值规则
- 实名认证与防沉迷的关系

### 应引用的关键规则
- 未成年用户仅可在周五、周六、周日和法定节假日 20:00-21:00 游戏。
- 12 岁以下禁止充值。
- 12-15 岁：单次 50 元、每月 200 元。
- 16-17 岁：单次 100 元、每月 400 元。

### 典型分流入口
- 查询实名：腾讯健康系统小程序 → 实名认证查询
- 修改实名：仅在系统显示有入口且满足条件时处理；未成年账号不支持随意修改
- 查看名下账号：腾讯健康系统小程序 → 近期登录的游戏账号 → 验证身份证查询

## 三、人脸识别相关问题怎么答

### 可以回答
- 为什么会再次出现人脸识别
- 人脸识别失败时的常见处理建议
- 人脸识别与账号实名本人使用的关系

### 不要回答
- 如何跳过人脸
- 如何取消系统弹出的人脸验证

### 可给出的官方建议
- 按页面提示由实名本人完成验证
- 更换时间段或设备重试
- 如多次失败，可拨打腾讯成长守护热线 0755-86013799（周一至周日 09:00-22:00）

## 四、成长守护内常见 FAQ 路由

### 1. 绑定/解绑/家庭关系
优先入口：
- 我的 → 家庭管理
- 我的 → 账号解除家庭关联申诉

适合处理：
- 怎么绑定孩子
- 怎么移出成员
- 孩子被误绑/成年人被误绑
- 如何阻止孩子解绑
- 为什么无法加入新家庭

### 2. 管控不生效 / 功能异常
优先判断：
1. 孩子是不是在用别的账号
2. 是否已经完成绑定与身份验证
3. 是否可能受网络或当前页面异常影响

如果仍不能解决：
- 引导至家长服务助手 / 家长柜台
- 或拨打腾讯成长守护热线 0755-86013799

### 3. 举报与反馈
优先入口：
- 服务与帮助 → 我要反馈
- 服务与帮助 → 未成年人游戏防沉迷举报

适合处理：
- 黑产实名举报
- 游戏实名问题
- 人脸识别问题
- 游戏弹窗限制
- 其他意见反馈

## 五、回答策略

### 情况 A：问题仍在成长守护能力范围内
处理方式：
- 先给功能解释
- 再给入口和操作路径
- 最后提醒验证/绑定/规则边界

### 情况 B：问题已超出成长守护直接处理范围，但有官方去处
处理方式：
- 明确说明本 skill 不能直接处理
- 不要停在“我不能回答”
- 必须告诉用户下一步去哪个入口

### 情况 C：问题涉及政策或高风险规避
处理方式：
- 只给规则说明和官方入口
- 不给规避建议
