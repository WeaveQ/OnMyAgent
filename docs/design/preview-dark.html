8 · Focus & Accessibility

Focus ring token (focus.ring-color, width 2px, offset 2px) + keyboard-navigation cheat sheet. Tab through the controls below to see the ring.

Focus ring on primitives
Keyboard cheat sheet
  • Tab ⇧ Tab — move focus in DOM order
  • Enter Space — activate
  • Esc — close dialog / popover / dropdown
  • ↑ ↓ — navigate menu / listbox / tabs
  • ⌘K — open command palette
  • Home End — first / last in list

All icon-only buttons must carry aria-label. Loading rows announce via aria-busy="true" on the parent, not per-icon labels.

9 · Do's & Don'ts (quick reference)

Do
  • Use dls-* semantic tokens.
  • Reach for atoms → composites → row primitives.
  • Even-scale typography only.
  • Add mac:titlebar-no-drag to titlebar / sidebar-header controls.
  • Route all user-visible strings through i18n.
Don't
  • No text-[Npx], rounded-[Npx], arbitrary hex in page JSX.
  • No shadow-* for component hierarchy.
  • No rounded-full on standard CTAs.
  • No any / as any / as unknown as.
  • No raw h-N w-N for icon buttons — use size="icon-*".

Z-layer stack

Overlays compose against a fixed 6-level ladder. The demo below stacks each layer visually — toast sits above dialog, which sits above popover, and so on.
