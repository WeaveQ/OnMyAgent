Questions, answered常见问题 这里有答案

Is OnMyAgent based on OpenCode?OnMyAgent 是基于 OpenCode 的吗？+
Yes. OnMyAgent is built on OpenCode as a local agent runtime foundation, then adds a desktop workspace for agent registry, work queue, sessions, logs, artifacts, skills, memory, and approvals. It is not a replacement for OpenCode. It is the workspace around it.是的。OnMyAgent 基于 OpenCode 作为本地 Agent 运行基础，并在外层增加桌面工作台，用来管理 Agent Registry、Work Queue、会话、日志、产物、Skills、Memory 和审批。它不是替代 OpenCode，而是围绕 OpenCode 和其他 Agent 工具的工作台。
OnMyAgent vs Codex / Claude Code / OpenCode?OnMyAgent 和 Codex / Claude Code / OpenCode 是什么关系？+
Codex / Claude Code / OpenCode are agents. OnMyAgent is the workspace and local Agent control console that manages them: registry, queue, sessions, logs, diffs, artifacts, and approvals all in one local app. We do not replace them; we make them easier to operate.Codex / Claude Code / OpenCode 是 Agent。OnMyAgent 是管理它们的工作台和本地 Agent 控制台：Agent Registry、队列、会话、日志、Diff、产物、审批，都在一个本地应用里。我们不替代它们，而是让它们更好用。
OnMyAgent vs LibreChat / Open WebUI / LobeChat?OnMyAgent 和 LibreChat / Open WebUI / LobeChat 有什么区别？+
Those are excellent chat clients / LLM web UIs. OnMyAgent manages CLI agents, MCP, Skills, Memory, sessions, work queues, and artifacts on your machine. Different category.它们是优秀的聊天客户端 / LLM Web UI。OnMyAgent 在你的机器上管理 CLI Agent、MCP、Skills、Memory、会话、Work Queue 和产物。不同品类。
OnMyAgent vs n8n / Dify / Zapier?和 n8n / Dify / Zapier 有什么区别？+
n8n / Dify / Zapier are cloud workflow automation platforms. OnMyAgent focuses on local-first agent work: your code, your machine, your model keys, your artifacts. Different category.它们是云端工作流自动化平台。OnMyAgent 聚焦本地优先的 Agent 工作：你的代码、你的机器、你的 Key、你的产物。不同品类。
Is it free? Is it really open source?免费吗？真的开源吗？+
Yes. OnMyAgent is planned as free and open source for individuals, developers, learning, research, and self-hosted personal use. Commercial or enterprise use may require a separate license. Final terms will be published in the repo before release.是的。OnMyAgent 计划对个人、开发者、学习研究和自托管个人使用免费开源。商业或企业使用可能需要单独授权。最终协议条款会在仓库发布前明确写清楚。
Relationship with WeaveQ?和 WeaveQ 什么关系？+
OnMyAgent is the open-source local AI workspace for individuals, and under the hood it works as a local Agent control plane. WeaveQ is the enterprise Worker delivery platform with deeper business surfaces. Same team, different scope.OnMyAgent 是面向个人的开源本地 AI 工作台，底层是一个本地 Agent 控制平面。WeaveQ 是企业 Worker 交付平台，具备更深的业务能力。同一团队，不同范围。
Can my team use it?团队可以用吗？+
OnMyAgent is free for individuals. Team support is planned for later, with shared workspaces, permissions, and organization-level audit opening through private beta.OnMyAgent 面向个人免费。团队能力会放在后续路线图中，通过私有 Beta 开放共享工作区、权限和组织级审计。
