# Code of Conduct

OnMyAgent is an open-source project for local-first AI agent work. We want the community to be useful, respectful, and safe for contributors with different backgrounds, experience levels, and perspectives.

## Our Standards

Examples of behavior that helps the project:

- Use welcoming and inclusive language.
- Assume good intent, but be open to correction.
- Give feedback that is specific, actionable, and respectful.
- Focus technical discussions on evidence, reproduction steps, and trade-offs.
- Respect maintainers' time by keeping issues and pull requests focused.

Examples of unacceptable behavior:

- Harassment, intimidation, threats, or personal attacks.
- Insulting, demeaning, or discriminatory language.
- Publishing private information without explicit permission.
- Repeated off-topic disruption after maintainers ask for it to stop.
- Abusive behavior around security reports, vulnerabilities, or user data.

## Scope

This code of conduct applies in project spaces, including GitHub issues, pull requests, discussions, documentation, community chat, and any project-related events or communications.

It also applies when someone represents the project publicly.

## Enforcement

Maintainers may take action when behavior harms contributors, users, or the project. Actions may include:

- Clarifying expectations privately or publicly.
- Editing, hiding, or deleting comments.
- Temporarily limiting participation.
- Blocking a contributor from project spaces.

Maintainers should apply enforcement consistently and proportionally.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the project maintainers. Use the subject `[OnMyAgent conduct] <short summary>` when reporting by email or private message.

Please include relevant links, screenshots, or context when safe to do so. Do not include private information that is not necessary for the report.
