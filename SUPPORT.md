# Support

OnMyAgent is an open-source local-first desktop project. Community support happens through GitHub issues and discussions once enabled for the repository.

## Where to Ask

- Bugs: open a GitHub issue with the bug template.
- Feature requests: open a GitHub issue with the feature template.
- Security vulnerabilities: do not open a public issue; follow `SECURITY.md`.
- Usage questions: use GitHub Discussions when available, or open a focused issue with reproduction details.

## Before Opening an Issue

Please include:

- OnMyAgent version or commit.
- Operating system and architecture.
- Whether you are using desktop, web dev mode, or headless mode.
- Relevant logs with secrets redacted.
- Reproduction steps.
- Expected behavior and actual behavior.

## Maintainer Expectations

This project is early and local-first. Maintainers may prioritize security, data safety, reproducibility, and core desktop workflows over broad feature requests.
